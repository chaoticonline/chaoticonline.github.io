Chaotic Changelog rev. fee56ab

Changelog

  rev. fee56ab

 

Added

Structure-related changes

  • Added an hourglass SVG icon to represent the Temporary Allegiance of some Creatures with the M’arrillians

  • Added the FrontMatter Parameter isGroup to the Listing Page-Files /perim/creatures/_index.<LANG>.md after all they also are a “categories’ category”

Hugo Templates-related changes

  • Added a top-margin to the Advertisement Block in Cards’ Search Tool Page

  • Implemented Hugo’s concept of Base Templating allowing to better organize all Template Files by fractionating big templates in smaller blocks per file. In this approach we’ve defined six blocks:

    • category** to set the HTML attribute id for main the <section> tag
    • subcategory to set additional CSS classes to be used in the main <section> tag
    • section to define the value for the <h2> tag
    • title to define the value for the <h3> tag
    • content for the main content (doh)
    • pagination used specifically by Listing Pages’ Templates to build the Pagination Links

  • Implemented the necessary blocks part of Hugo’s Base Templating in the Template Files for Listings and Single Pages, for both Default and for Videos, refactoring and re-indenting whenever it was possible — i.e. avoiding multiple else statements to build a determined structure by previously creating the said structure with default values

  • Implemented a way of showing Advertisement Blocks in all but Creatures’ Tribes Listing Pages regardless the viewport by reducing the number of results per Page by one and removing the control class that was hiding it. Because we’re loading less one result, Pages in which all results have a proper Big Picture — thus filling all available columns — will load slightly faster because there will be one less image to download

Public Content-related changes

  • Added a second “in-ads” advertisement block in the published Article about Tangath Toborn since it is very long and thus comport without forcing too much

  • Published Episode #25, Season #01 with Brazilian Portuguese audio

CSS-related changes

  • Added the necessary CSS rules as a counterpart of the hourglass SVG icon added

  • Added a few more CSS rules to Advertisements Blocks as a counterpart of the revamped implementation to have them mixed together the results of a Listing Page (see above)

Removed

Structure-related changes
  • With the integration of the Data Files of Creatures/Locations appearances in the Animation with the Data Files holding the majority of their information, the old ones located at /data/tv have been removed
Public Content-related changes
  • Removed a wrongly added call to the Creatures (TV) Shortcode in the componentizable published Article about Lava Pond, Milla’iin’s Foothold (Brazilian Portuguese only)
CSS-related changes
  • Removed unused and unnecessary rules

Changed

Structure-related changes
  • Since Hugo’s Developers themselves recognize the way they handle JSON Data Files isn’t the most optimal, consuming a lot of memory, we’ve decided to integrate the list of Episodes in which a Creature/Location (and, in the future, Battlegears and Mugics as well) appear in the Animation with the Data Files holding the majority of their information (name, rarity…). Even though we’re duplicating the same information among all localized files (i.e. for every available language), at least we can try to reduce the memory usage, especially during the development
Hugo Templates-related changes

  • Revamped the implementation of the Partial Template for Google AdSense adding a debugging border for when running the site through Hugo’s Server or if the Development Parameter is set to true in config.yaml file -AND- to automatically add the Client ID from the newly added configuration option when the proper Parameter/Attribute hasn’t been set in the Shortcode call

  • While implementing Hugo’s concept of Base Templating, we separated Cards’ Listing Pages from Videos Listing Pages, reducing the logic necessary for each Template, even though most of the code between them is identical

  • Adjusted the main text colour added inline in the Header Partial Template as part of the Critical Path methodology to match the Colour Contrast Optimization made a few revisions ago

  • Refactored the Perim Partial Template responsible to build the Single Page for Cards’ Articles by removing Hugo’s Scratch and improving the overall code whenever it was possible

  • Refactored the Creatures/Locations Shortcode Partials responsible to build the Cards’ Information Box by applying part of Hugo’s Base Templating concept and, in some cases, reusing relatively long logics multiple times

  • Refactored the Creatures/Locations (TV) Shortcode Partials responsible to build the list of Episodes in which they appearaed and, in case of Creatures, the box with their Voice Actors

Public Content-related changes

  • With the revamp in Google AdSense Partial Template *ALL* non-componentizable published Articles have been updated to simplify their calls to the Shortcode

  • Updated Changelog Page

CSS-related changes

  • Made some micro-adjustments to the CSS ribbons used show Episodes Numbers on their Listing Pages which got misaligned after implementing the palliative solution to prevent Google Chrome to show unwanted white borders

  • With the revamped implementation of Listing Pages (see above) all CSS classes named .listing has been converted to an ID as in #listing

  • With the revamped implementation of Listing Pages (see above) all CSS classes named .page.videos has been converted to an ID as in #videos

  • Reduced the padding of Card Blocks of Listing Pages results

  • Reworked the CSS definitions of Listing Pages’ Thumbnails as a counterpart of the revamped implementation made (see above)

Fixed

Structure-related changes
  • Fixed a long-term issue with Responsive AdSense blocks yielding a Console errors regarding the available width of their parent element by deferring their inclusion to the global variable adsbygoogle only when the DOM Tree is ready
Public Content-related changes
  • Fixed capitalization of Rodrigo Andreatto in the Cast of Episode #25, Season #01 (Brazilian Portuguese only)
CSS-related changes
  • Fixed an issue with the horizontal lines crossing the “Read More” box in Changelog Page which, sometimes, simply disappear from some of them

Image-related Changes

  • Resized the Big Pictures of Seasons #2 and #3, shrinking them to 1920x1080, the maximum dimensions for Thumbnails/Images in XML Sitemap for Videos. Because of the wrong sizing, the proper tags <thumbnail_loc> weren’t being added to the XML, thus resulting in a relatively invalid markup

Miscellaneous changes

  • Updated (again, again) all XML Sitemaps with a different namespace declaration as (another) attempt to make Google Crawl them. This time, however, before sending them to Google Console, each one of them have been tested individually and Google didn’t report any error

  • Updated minified CSS and JS
