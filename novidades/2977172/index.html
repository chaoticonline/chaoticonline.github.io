Adições

Estruturais

  • Aplicada Componentização de Arquivos de Conteúdo para a Página de Novidades com entradas i18n e CSS/JS adicionais para ocultar a maior parte do conteúdo em blocos retráteis

  • Adicionada nosso Perfil no Github como outro possível valor para a Propriedade sameAs para a Entidade Organization do Schema.org

  • Adicionados Arquivos de Conteúdo, Data Files e entradas i18n para as seguintes Páginas como contraparte das referências feitas no Artigo sobre o Pilar de Ferro publicado nesta revisão: Gigantroper (Equipamento de Batalha) e Pilar de Gelo, Pilar de Jade, Pilar Entalhado, Pilar de Vapor, Pilar de Pedra e Pilar de Madeira (Locais)

  • Adicionada uma forma ainda melhor para se mostrar o texto expandido das tags <abbr> em dispositivos móveis

Relacionados ao Hugo
  • Adicionado atributo HTML nofollow aos links apontando para as Páginas de Termos de Uso e Novidades em uma tentativa de não ter o Google considerando-as como fontes mais referenciadas apenas porque elas aparecem no rodapé de todas as Páginas sendo que o mais importante é ter Páginas interconectadas (ex: Artigos referenciando um Episódio)
Relacionados ao Conteúdo Publicado

  • Adicionadas palavras-chave aos seguintes Artigos componentizáveis publicados: Maxxor, Protetor de Perim; Najarin, Inimigo dos Fluidmorphers; Najarin, Supremo Muge do Lago; e Najarin, Jovem

  • Adicionado o Parâmetro de FrontMatter datePublished ausente nas seguintes Páginas/Artigos publicados: Bodal; Bodal, Pesquisador da Broca Flamejante; Maxxor, Protetor de Perim; Najarin, Inimigo dos Fluidmorphers; Najarin, Supremo Muge do Lago; Najarin, Jovem; Magmon; Magmon, Vingador; Magmon, Incorporado; Takinom, Amazona das Sombras; Zhade; Wamma; Wamma, Auxiliar de Artilharia da Colmeia; Wamma, Sentinela das Chamas; e Novidades

  • Publicado Artigo sobre o Pilar de Ferro

Remoções

Estruturais
  • Com a implementação da Minificação de Recursos do Hugo na revisão anterior, não há mais necessidade de se ignorar nada e, por isso, o arquivo .gitignore foi removido

Alterações

Estruturais

  • Alterados os identificadores i18n das chamadas das Páginas de Cartas para serem mais consistentes com a nova e melhorada implementação de SEO

  • Atualizados códigos de acesso ao Bing e ao Yandex

Relacionados ao Hugo

  • Modificado o Shortcode Include para que não exija um arquivo se o formato requisitado for Git Hash, “lançando um erro” se o GIT HEAD não existir caso o repositório esteja corrompido ou sequer exista. Isso porque o propósito desse recurso, no momento, é apenas obter o hash da revisão anterior para preencher os Títulos de Seção da Página de Novidades

  • Modificado o Template Parcial de Navegação para deixar que a largura da coluna do Alternador de Idiomas seja gerenciada pelo próprio Bootstrap, o que nos permitiu adicionar um espaçamento entre o Logo do Site e o Menu de Navegação

  • Refeita a implementação SEO de, inicialmente, todas as Páginas de Cartas através de alguns pequenos Templates Parciais componentizáveis para cada diretiva, oferecendo uma grande manutenibilidade do código

Relacionados ao Conteúdo Publicado

  • Foram feitas adições e correções na redação e revisão dos links dos Artigos publicados sobre o Maxxor e Najarin

  • Renomeada a versão em Português Brasileiro do Castelo dos Pilares para Pilar do Castelo porque, mesmo que diferente da versão com áudio dublado (de primeira aparição no Episódio #22, Temporada #01), soa mais correto, considerando que o Pilar do Castelo é um dos sete pilares segurando o teto do Submundo, em vez de um Castelo feito de Pilares

  • Com a renomeação da versão em Português Brasileiro do Pilar do Castelo, todas as ocorrências do nome antigo foram alteradas ao longo do site

  • Atualizada Página de Novidades

Relacionados aos Estilos do Site
  • Aumentou-se um pouquinho a distância do Alternador de Idiomas da borda direita, pois o texto do idioma ativo estava sendo cortado por um breve momento durante a animação de recolhimento

Correções

Estruturais

  • Corrigido um problema com o novo arquivo CSS combinado a partir de fontes componentizáveis individuais introduzido na revisão anterior que ainda estava incluindo a família de fontes usada exclusivamente pela Ferramenta de Busca de Cartas em todos os arquivos, pois as instruções estavam erroneamente duplicadas junto da fonte principal usada (Quicksand)

  • Corrigido um problema com os Dados Estruturados para Vídeos no qual as miniaturas dos Episódios não estavam sendo adicionadas, pois a estrutura correspondente não foi atualizada quando boa parte dos Parâmetros de FrontMatter foi movida para os Data Files do Hugo

Relacionados ao Hugo

  • Corrigidos “erros” de sintaxe nos Templates de Listagem e Páginas Individuais e o Template Parcial dos Breadcrumbs que, mesmo que estes funcionassem perfeitamente com a versão 0.47 do Hugo e que a funcionalidade da mudança nem mesmo esteja em uso, não funcionava mais na versão atualizada do Hugo usando a versão atualizada do GO

  • Corrigido um problema que impedia as Páginas de Listagem de Vídeos de terem miniaturas

Relacionados ao Conteúdo Publicado
  • Corrigidos todos os Arquivos de Conteúdo de Curiosidades para que a correção feita há algumas revisões atrás para trazer de volta os Resumos pudessem ser aplicadas neles também
Relacionados aos Estilos do Site
  • Corrigido o Menu de Navegação retrátil que ficou todo bagunçado na versão mobile após algumas alterações feitas na revisão anterior

Relacionadas às Imagens

  • Adicionada uma versão menor do Logotipo do Site para ser usado nos Dados Estruturados para SEO. Não parece bom, mas se o Google manda na Internet e diz que é assim…

  • Adicionadas Imagens Grandes para as seguintes Páginas adicionadas como contraparte das referências feitas no Artigo sobre o Pilar de Ferro publicado nesta revisão: Pilar de Jade, Pilar de Pedra e Pilar de Madeira (Locais)

  • Adicionadas Imagem de Carta para o Pilar de Ferro como complemento de seu Artigo publicado nesta revisão

Alterações Diversas

  • Atualizados os arquivos de Sitemap para Equipamentos de Batalha, Locais e Índice

  • Atualizados CSS e JS minificados
