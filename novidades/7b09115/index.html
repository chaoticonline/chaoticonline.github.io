Adições

Estruturais

  • Adicionado um valor padrão paras a META Tag description caso não tenha sido definido um Resumo, o texto antes da marcação especial more do Hugo, -E- se após substituir todas as quebras de linha do HTML (<br/>) por novas linhas (\n) o Resumo resultante for uma string vazia ou apenas com caracteres não imprimíveis (ex: vários espaços). Dessa forma nós garantimos que todas as Páginas, mesmo aquelas que ainda não tenham sido efetivamente publicadas ainda por ausência de conteúdo tenha um valor para esta tag

  • Adicionado arquivo XML de autorização do Bing para que este possa indexar devidamente o site porque, ao que parece, a Microsoft não entende que, hoje, aspas em torno dos valores dos atributos HTML dessas tags e/ou a barra de encerramento dessas tags com auto-encerramento são opcionais

Relacionados ao Conteúdo Publicado

  • Adicionado o Parâmetro de FrontMatter publishingDate ausente nos Artigos publicados sobre o Magmon (Criatura), Tanque de Lava e Pilar de Ferro (Locais) e cada uma das revisões componentizáveis na Página de Novidades, mesmo que tal informação não esteja sendo adicionada nela enquanto a nova implementação de SEO não seja aplicada as Páginas de Vídeos por conta de conflitos com Parâmetros de FrontMatter entre eles

  • Adicionado o Parâmetro de FrontMatter group ausente nos seguintes Artigos componentizáveis publicados: Bodal, Pesquisador da Broca Flamejante; Maxxor, Protetor de Perim; Najarin, Supremo Muge do Lago; Najarin, Jovem; Takinom, Amazona das Sombras; Wamma, Sentinela das Chamas; e Wamma, Auxiliar de Artilharia da Colônia

  • Adicionado o Parâmetro de FrontMatter group ausente no Artigo componentizável publicado sobre Najarin, Inimigo dos Fluidmorphers (apenas Português Brasileiro)

  • Adicionado os Parâmetros de FrontMatter publishingDate -E- group ausente nos seguintes Artigos componentizáveis publicados: Geleiras, Geleiras, Campo de Derretimento M’arrillian; Canhão de Calor M’arrillian; Lago Ken-I-Po, Local do Ritual; Região dos Rios, Resíduos Abandonados; Região dos Rios, Base de Operações de Erak’tabb; Cidade do Submundo, Durante a Ascensão de Van Bloot; e Tanque de Lava, Base de Operações de Milla’iin (Locais) e Magmon, Vingador; e Magmon, Incorporado (Criaturas)
