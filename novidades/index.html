Alterações

Estruturais
  • Com a nova e melhorada forma de se incluir arquivos CSS e JS (ver abaixo), todos os arquivos de /themes/chaotic/src foram movidos para /assets, o diretório padrão dos Recursos compilados do Hugo, o que, de qualquer jeito, significa a remoção e a adição de arquivos
Relacionados ao Hugo

  • Alterada a forma como arquivos CSS são incluídos por fazer uso da metodologia Critical Path, carregando todos os estilos necessários depois que tudo na página tiver sido renderizado, tornando o carregamento mais rápido a partir da segunda visita ao site — quando o navegador já tiver feito cache dos estilos — ao custo de ter elementos piscando ou mesmo completamente sem estilos (conexões lentas) até a página ser totalmente carregada

  • Refeita a forma como arquivos CSS e JS são incluídos por expandir a funcionalidade do Parâmetro de FrontMatter include atualmente em uso apenas pela Ferramenta de Busca de Cartas. Com esta importante mudança, não apenas já contamos com uma pequena melhora na performance por não ter uma das fontes usadas exclusivamente pela Ferramenta de Busca de Cartas sendo carregada em toda a parte, mas também em versões futuras isso nos permitirá fragmentar os arquivos CSS atualmente em uso e, assim, carregar apenas aquilo que realmente é necessário para cada Página. Por exemplo: Aquelas faixas adicionadas no canto superior esquerdo de cada miniatura nas Páginas de Listagem de Episódios não são necessárias em nenhuma das Páginas de Listagem de nenhuma seção sob o menu Perim, então não carregaremos mais as instruções para que tais faixas funcionem. Da mesma forma, os ícones de Disciplinas e Afinidade Elemental não são necessários em nenhuma das Páginas de Vídeos, então não os carregaremos lá também

Relacionados ao Conteúdo Publicado

  • Ajustados todos os Analisadores da ferramenta de Busca de Cartas para aplicar a correção feita na estrutura HTML dos Templates Parciais dos Tipos de Cartas (ver abaixo)

Relacionados aos Estilos do Site

  • Melhorado o alinhamento vertical do botão de alternância de visibilidade no Menu de Navegação em dispositivos móveis

  • Reduzida a altura das miniaturas em dispositivos móveis

Correções

Relacionados ao Hugo
  • Corrigida a estrutura do HTML dos Templates Parciais para Criaturas, Locais e Mugics para usar o atributo HTML class em vez de id porque mesmo que com a Componentização de Arquivos de Conteúdo os Artigos terão de fato apenas um id seus Artigos principais terão mais de um, resultando, assim, não apenas em HTML com marcação semanticamente incorreta, mas também interferindo no funcionamento da Ferramenta de Busca de Cartas
Relacionados ao Conteúdo Publicado
  • Corrigido um problema com os Analisadores da Ferramenta de Busca de Cartas de Ataques, Equipamentos de Batalha e “Todos” que buscavam pela raridade da Carta sem especificar que apenas a primeira ocorrência deveria ser usada
Relacionados aos Estilos do Site

Agora a correção de rolagem horizontal será aplicada APENAS em resoluções menores em vez de globalmente
