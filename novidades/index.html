Mudanças de Versão do site Chaotic Online

Versão Atual

Publicada em 02/07/2018 às 10:30:00

Adições

Estruturais

  • Adicionada a Entidade HTML &nbsp; a quase todas as tags vazias para que elas não sejam de fato vazias e, assim, consideradas como possíveis candidatos a remoção por HTML Minifiers

  • Adicionada uma mensagem de erro diferente ao Buscador de Cartas, permitindo que um dos dois possíveis erros fatais que impedem os resultados de uma busca de aparecer possam ser distinguidos

  • Adicionada uma entrada ao .gitignore removida erroneamente em commits anteriores

  • Adicionado Dados Estruturados de SiteNavigationElement apontando para o Menu de Navegação como mais uma melhoria para o SEO (Otimização para Motores de Busca)

Sobre as Páginas de Vídeo
  • Publicado Episódio #15, Temporada #01 com áudio dublado em Português Brasileiro
Sobre o Hugo

  • Adicionado/Ajustados várias classes do Sistema de Grade do Bootstrap nos Arquivos de Modelo para prover responsividade máxima em diferentes dispositivos e/ou resoluções

  • Adicionada a Propriedade inLanguage aos Dados Estruturados Dados Vinculados do Schema.org

  • Adicionado um elemento <wrapper> que engloba todo o conteúdo das páginas do site, como parte da correção do rodapé cobrindo parte do conteúdo principal. A técnica foi emprestada deste Codeply e, apesar de funcionar, a presença de uma tag <wrapper> ali torna semanticamente inválido

Sobre os Estilos

  • Adicionados os ícones do Github e do HUGO, anteriormente adicionados ao arquivo com os ícones SVG da UI, ao novo rodapé

  • Adicionadas mais Media Queries como auxílio a melhor responsividade criada com as alterações inclusões das classes do Sistema de Grade do Bootstrap

Sobre Conteúdo Público

  • Adicionada apreciação à Equipe do VERO na página de Dizeres Legais

  • Adicionado essa página com informações para nerds (:p) sobre o que mudou de uma versão pra outra

Remoções

Estruturais
  • Removida entrada i18n (Internacionalização) para o link social do Patreon. Não apenas porque ele já está oculto há algum tempo mas também porque a nova marcação de links sociais (ver abaixo) não usa nenhum texto
Sobre o Hugo

  • Removidas restrições de zoom

  • Removidos alguns usos da função absURL do Hugo em favor de URLs Relativas

  • Removidas indentações desnecessárias em todos Modelos Parciais, afinal, eles serão usados de forma composta, mas editados individualmente

Sobre os Estilos

  • Removia a margem superior usada nos Ícones das Bandeiras porque o novo Menu de Navegação tem um melhor alinhamento graças ao Flexbox usado

  • Com todos os novos recursos e alterações, VÁRIAS regras não são mais necessárias e foram removidas. Um total de aproximadamente 300 linhas removidas

Alterações

Estruturais
  • Alteradas algumas URL desnecessariamente Absolutas em favor de Relativas
Sobre o Hugo

  • Os Dados Estruturados para Organização foi movido para o topo, ficando de um jeito mais hierárquico

  • A tag <aside> do lado esquerdo, usada para os links sociais, foi movida para fora do conteúdo principal, como deveria

  • A tags <div> usada como encapsulamento do conteúdo principal foi substituída por <main>

  • O Modelo Parcial de Navegação foi alterado por outro mais limpo quase sem nenhum ajuste no CSS o que traz também uma melhor responsividade para dispositivos móveis

  • O Modelo Parcial de Rodapé foi refeito com uma versão nova e mais responsiva, removendo, assim, a marcação adicional específica para dispositivos móveis

  • A marcação da Imagem Grande foi modificada a fim de remover a marcação específica para dispositivos móveis

  • A marcação dos Links Sociais foi alterada por uma mais simples. Embora não tão atraente, que abre/fecha com o passar do mouse, essa versão não precisa de nenhuma gambiarra apara acomodar textos de diferentes comprimentos — ex: o rótulo do agora oculto Patreon

  • A marcação das Imagens das Cartas foi modificada por um melhor alinhamento vertical -E- para finalmente remover a “gambiarra” usada há muito tempo para lidar com o problema de “canto arredondado branco” no Google Chrome

  • As marcações de Seção e Títulos de Artigos foram atualizadas a fim de remover a marcação específica para dispositivos móveis

  • Com a remoção da tag <aside> de dentro do conteúdo principal, a largura da coluna para dispositivos maiores (largura >= 996px) foi aumentada

  • Alterada a classe CSS usada em um dos dois possíveis erros fatais que impedem os resultados de uma busca de aparecer possam ser distinguidos

Sobre os Estilos

  • O novo Menu de Navegação fez o Flyout Content — a caixa que abre ao passar o mouse — do Menu de Idiomas muito pequeno, então algum espaçamento foi adicionado a eles

  • Com as melhorias no alinhamento vertical das Imagens das Cartas, os estilos de dos Flavour Texts em forma de Citação, seus elementos internos e próximos (ex: Índice) tiveram que ser ajustados também pois alguns Artigos, de algumas Cartas, os tem logo abaixo de uma Imagem de Carta — ex: Najarin

  • Ajustada a margem superior do Ícone de Carregamento

  • Ajustadas as margens da mensagem do contêiner dos Provedores de Vídeo disponíveis bem como da própria mensagem de Seleção dos Provedores disponíveis

  • Todas as regras necessárias para todas as adicções/mudanças feitas nessa revisão foram adicionadas, alteradas ou removidas

  • Algumas classes foram reordenadas de uma forma mais pseudo-hierárquica

Sobre Conteúdo Público

  • Aplicadas correções orto-gramaticais à versão em Português Brasileiros dos seguintes Artigos/Páginas: Maxxor, Najarin, Blazier, Heptadd, Takinom, Nauthilax, H’earring, Chaor, Malvadine, Geleiras, Cidade do Submundo, Caverna de Cristal, Região dos Rios e as Páginas de Dizeres Legais e Termos de Uso

  • Aplicadas correções gramaticais à versão em Inglês das Páginas de Dizeres Legais e Termos de Uso

Alterações

Estruturais

  • Alterado o Parâmetro de FrontMatter thumbnail por picture, afinal, ele cuida mais da Grande Imagem dos Artigos do que as Miniaturas das Páginas de Listagem

  • Alterados o Parâmetro de FrontMatter group para section, porque apesar de não influenciar a variável .Section do Hugo é nome mais apropriado para usar

  • Renomeado o Parâmetro de FrontMatter languageMirrors para languages, que é mais limpo e combina melhor com o conceito

  • Alterada a forma como são gerenciados os Parâmetros de FrontMatter eyes, hair e skin, todos agrupados sob o Parâmetro traits. Agora, em vez de terem múltiplos valores separados por uma barra manualmente adicionada, ele serão uma um array indexado que serão automaticamente renderizados em forma de lista, assim como outros Parâmetros que aceitam múltiplos valores

  • Alterada aforma como estilos CSS customizados são definidos na Imagens dos Artigos. Em vez de todos eles serem definidos numa única linha, como uma declaração inline, Agora eles serão definidos como um array associativo de dois índices, regra e valor. Com essa mudança, nós podemos filtrar regras que poderiam causar comportamentos indesejados, especialmente quando o site é acessado por um smartphone (ex: height e max-height)

  • Refletidas as renomeações dos Parâmetros de FrontMatter mencionadas em todos os Arquivos de Conteúdo

  • Migrados todos os Créditos de Imagem da Página de Dizeres Legais para cada imagem, Grandes ou de Cartas, em cada Artigo

  • Com a remoção do Título de Seção “Habilidades” mencionado acima, todas as variações de Cartas agora estão um nível mais adentro na hierarquia do Índice, caindo sob “Jogo”

  • Renomeado o Arquivo de Conteúdo da Dranakis Threshold para combinar com a convenção de nomenclatura de todos os outros. De qualquer jeito, isso significa a remoção e a adição de arquivos

Sobre os Artigos

  • Revisados todos os Links em Markdown, adicionando uma barra (/) ao final de cada um deles

  • Alterada a entrada skin sob o Parâmetro de FrontMatter trait de Blazier para a tradução em Português do Brasil, casando melhor com “Amarelo Girassol” da versão em Inglês

  • Alterada a interpretação da expressão Mugicians em todos os Artigos. Agora eles são descritos apenas como “Mugicians” em vez de “Contadores de Mugicians”, representando-os como as formas espirituais dos criadores de Mugic do passado, que coexistem com as Criaturas e energizam seus Mugics

  • Foram feitas correções orto-gramaticais, adições e correções na redação nos seguintes Artigos: Maxxor, Najarin, Blazier, Heptadd, Takinom, Nauthilax, H’earring (somente Português Brasileiro), Chaor, Malvadine, Geleiras, Caverna de Cristal, Lago Ken-I-Po, Região dos Rios e Cidade do Submundo

  • Atualizada a Página de Dizeres Legais para também agradecer o pessoal por trás do Grammarly

  • Atualizado as descrições dos Parâmetros de FrontMatter’ na versão em Inglês do Artigo da Takinom

Sobre as Páginas de Vídeo

  • Todos os espelhos no YouTube agora são forçados a usar a versão mais de acordo com a GDPR por alterar o domínio para youtube-nocookie.com

  • Alterada a Data de Publicação de todos os Episódios dublados em Português do Brasil

Sobre o Hugo

  • Ajustadas diversas classes do Sistema de Grade do Bootstrap e adicionadas algumas CSS Media Queries para melhorar a legibilidade e a disposição do conteúdo em dispositivos móveis

  • A tag <script> do Google AdSense foi movida do Modelo Parcial de Cabeçalho para seu próprio sob o subdiretório /google, junto com o agora renomeado google.html para analytics.html, o que, de qualquer jeito, significa a remoção e a adição de arquivos

  • O SDK do Facebook e o Modelo Parcial com as tags do Plugin de Comentário foram movidas do subdiretório /social para o diretório principal dos Modelos Parciais nomeado como facebook, seguindo a convenção de nomenclatura aplica à Família Google, o que, de qualquer jeito, significa a remoção e a adição de arquivos

  • O Modelo Parcial dos Links Sociais foi movido do subdiretório /social para o diretório principal de Modelos Parciais e renomeado para social.html, o que, qualquer jeito, significa a remoção e a adição de arquivos

  • A longa e de difícil leitura condição para os Anúncios na Barra Lateral foi quebrada em duas condições aninhadas

  • O JS/HTML dos Modelos Parciais usados foram expandidos, afinal, se eles serão comprimidos novamente por algum HTML Minifier ou pela Cloudflare, que fiquem legíveis

  • O Shortcode de “Citações” foi simplificado com uma versão mais limpa e com uma marcação menos problemática que usa os Atributos de um Shortcode do Hugo em vez dos Parâmetros de FrontMatter, permitindo que um único Modelo de Shortcode gerencie todos os casos de uma Citação, mesmo que se houver um que fique abaixo de uma Imagem Grande precisem agora ser adicionado manualmente nos Arquivos de Conteúdo

  • A marcação das Grandes Imagens foi modificada para refletir as alterações feitas no CSS para acomodar os Créditos das Imagens

  • A “gambiarra” usada para verificação de existência de arquivos foi substituída pela função fileExists oferecida pelas novas versões do Hugo

  • A coluna com a Biografia das Criaturas foi aumentada para visualização em dispositivos móveis por reduzir um pouco a coluna com os rótulos dando espaço para mais texto

  • A marcação do Shortcode “Picture” foi expandida para refletir as mudanças feitas no CSS para acomodar os Créditos das Imagens -E- para melhorar o visual com margens agora universalizadas

  • A marcação dos Episódios/Curiosidades foi ajustada para refletir as mudanças feitas no CSS para melhor acomodar os Breadcrumbs

  • O Rodapé foi refeito com uma estrutura melhor agora com os links legais, mensagem de copyright e links com as ferramentas usadas

  • Basicamente todo o conteúdo de todos os Modelos usados para os Episódios/Curiosidades foram re-indentados
