Changelog

  Current Version

 

Added

Structure-related changes

  • Added Content Files, Data Files and i18n entries for the following Pages as a counterpart of the references made in the Article about Kiru City published in this revision: The OverWorld Library and The Training Grounds (Locations); and Drabe, Hune Paltanin and Raznus, Assimilated (Creatures)

  • Added ESLint’s Configuration file

Public Content-related changes
  • Published Article about Kiru City

Removed

Structure-related changes
  • Given the update of Babel, we decided to remove the package.json file from the list of ignored files on .gitignore
Hugo Templates-related changes

  • Removed the loop over the Page entry of the Local Scratch for SEO just to create the META Tags description and headline, writing them manually instead

  • Removed a few other unused entries from .gitignore file

Public Content-related changes
  • Removed Kiru City as one of the keywords from Episode #06, Season #03 because the city itself didn’t even exist back in the days of Ancient Perim

Changed

Structure-related changes

  • Revised all JavaScript code accordingly to ESLint’s suggestions

  • The “fix” regarding Google AdSense blocks has been reverted because deferring them to the document readiness made them stop working

  • Updated Babel version

Hugo Templates-related changes

  • Finished the optimization of Hugo’s Scratch, replacing them for basic array/slice whenever it was possible

  • Simplified the Header Partial Template to not pass the Paginator stored in a global Scratch as a parameter for the main SEO Partial Template as this is unnecessary

  • Stored the arguments passed to all smaller Partial Templates for SEO in a Template variable, reusing Hugo’s dict

  • Expanded Facebook Comments Plugin Partial Template to NOT add the required markup on Listing Pages. The SDK Partial Template, however, remains untouched because the Facebook SDK is needed for other resources provided by the company

Public Content-related changes

  • Added the additional entries card and appearances to the Data File of Kiru City with, respectively, the Card Image path and the List of Episodes in which the City appeared in the Animation

  • Updated Changelog Page

CSS-related changes

  • Changed the main background-color in both the externa CSS file and the inline declaration of the Critical Path Methodoloy, to a very subtle tone of grey. It’s almost imperceptible, but it helps reading coloured links like those of Locations or Tribeless’ Creatures

  • With the change above, we changed the background-color of the clickable Read More/Less boxes of the Changelog Page as well so they don’t look weird

Fixed

Structure-related changes

  • Fixed, with a palliative solution, the issue involving Hugo’s minifier not removing doc-blocks from JavaScript files entirely, leaving random asterisks that could result in syntax errors

  • Fixed an issue with the Data File of Raznus, Ambassador to the Danians which had the path for his Big Picture pointing to Olkiex’s and since there’s no Big Picture for that Creature (yet?), Raznus’ Page had none as well

Public Content-related changes

  • Fixed an issue with Cards’ Search Tool not returning anything when choosing Generic as a filter for Tribe while Card Type had been defined as Mugics

  • Fixed the Brazilian Voice Actor of Vidav previously defined as Faduli Costha while the correct would be Sidney Lilla

Image-related Changes

  • Replaced the Big Picture of Kiru City (Location) with a version of slightly better quality (to be nice <_<)

  • Added Big Picture for the recently added Page about The OverWorld Library (Location) as a counterpart of the reference made in the Article about Kiru City published in this revision

  • Added a Card Image for Kiru City, as a counterpart for the Article published in this revision

Miscellaneous changes

  • Updated XML Sitemaps for Locations, Creatures and Index

  • Updated minified CSS and JS
