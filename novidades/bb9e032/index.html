Chaotic Changelog rev. bb9e032

Changelog

  rev. bb9e032

 

Added

Structure-related changes

  • Added Content Files, Data Files and i18n entries for the following Pages as a counterpart of the references made in the Article about Tangath Toborn published in this revision: Deadwater Devastation, Degenervate, Fearless Strike, Grounding Grapple, Primal Smash and Unified Tribecharge (Attacks); Burithean Axe (Battlegear); and Aimukk; Faash; Gorram, Danian General; Tassanil; Drimesse; Fivarth; Ghatup; Savell; Nebres; Targubaj; Viqtarr; Lyssta; Rothar and Ultadur (Creatures)

  • Added missing i18n entry for Tartarek

Hugo Templates-related changes

  • Added caching functionality to the Footer Partial Template of Home Page, Taxonomy Terms Page and Videos Single Page, as they don’t need custom CSS/JS

  • Added top-margin for Cards’ Search Tool error messages

  • Added NO-INDEX Meta Tag to Changelog Pages (and its Revisions) and Terms of Use Pages in order to prevent these pages to get indexed (obviously) but also as an attempt to prevent links and expressions on these Pages to be wrongly counted as “top linking text”

Public Content-related changes

  • Added a slightly humorous paragraph to the Cards’ Search Tool explaining why sometimes a query certain to return at least one result could bring none, suggesting the User click the button again. This problem occurs with fast connections in which, even if all AJAX Requests needed are successfully completed even if requesting — for example, data from all Creatures (more stuff to parse) — everything happens so fast that the FooTable plugin might not react in able time. By clicking a second time the button, all requests have already been made e the proper elements filled with all data, leaving only the Plugin to create the dynamic table

  • Published Article about Tangath Toborn

Removed

Structure-related changes

  • Removed a relatively duplicated i18n identifier for the Brazilian Voice Actor Mauro Eduardo. Relatively because only the entry with its Profile at MyAnimeList was duplicated, one of the entries had its full name while the other only first and last names

  • With the merging of Creatures’ Information Data Files with their TV Appearances (see below), the old ones located at /data/tv/dubbers have been removed

  • With the merging of Creatures’ Data Files described above, there was no need of another sub-level under /data/tv just for Creatures/Locations appearances in the Animation, so we removed it pulling the directory structure inside one level above, which, by all means, represents files removal and the inclusion of new ones

Public Content-related changes
  • Removed period from some paragraphs of old Changelog’s Revisions <_<

Changed

Structure-related changes
  • Merged Creatures’ Voice Actors Data Files with the main version of Creatures’ Information Data Files (i.e. tangath-toborn.en.json), thus having fewer JSON files to be read
Hugo Templates-related changes

  • With the merging of Creatures’ Data Files (see above), the Partial Template for Creatures (TV) has been adjusted to reflect the changes

  • With the removal of the additional directory for Creatures/Locations appearances (see above), the Partial Templates for Creatures (TV) and Locations (TV) have been adjusted to reflect the changes

Public Content-related changes

  • With the removal of the i18n identifier for Mauro Eduardo in favour of the one with his full name, all occurrences using the old format have been updated throughout the site (Brazilian Portuguese only)

  • Revised all internal links to add a trailing slash to minimize the redirection need

  • Updated Facebook SDK version

  • Updated Changelog Page

Fixed

Structure-related changes

  • Fixed an issue with the i18n file of Creatures in which, because of typo in the identifier of Wamma, Hive Ordnance and Wamma, Flame Sentinel, invalid URLs were rendered because the URL identifier, was overwriting this one with the name when dynamically adding anchors to texts (Brazilian Portuguese only)

  • Fixed XML attribute hreflang for Nauthilax, Tainted and Nauthilax, Aquatic Spy in the XML Sitemap for Creatures (Brazilian Portuguese only)

Public Content-related changes

  • Fixed capitalization of Rodrigo Andreatto in the Cast of Episode #24, Season #01 (Brazilian Portuguese only)

  • Fixed capitalization of Skartalas name in his Data Files

  • Fixed the Brazilian Portuguese Data File of *Wamma, Hive Ordnance whose name was in the wrong locale

Image-related Changes

  • Replaced the Big Picture of Frost Blight with a version of better quality

  • Replaced the Big Picture of Tangath Toborn with a version of better quality of his original form

  • Added the following Big Pictures referenced in the Article about Tangath Toborn published in this revision: Degenervate (Attack), Drimesse, Nebres, Targubaj, Viqtarr, Rothar and Ultadur (Creatures)

  • Added a Card Image for Tangath Toborn and his variations In Training and OverWorld General, as a counterpart for the Article published in this revision

Miscellaneous changes

  • Simplified the Sitemap Index Namespace declaration in an attempt of making Google find and parse all XML files listed

  • Updated XML Sitemaps for Attacks, Battlegears, Creatures and Index

  • Updated minified CSS and JS
