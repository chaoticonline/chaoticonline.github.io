Added

Structure-related changes

  • Applied Content File Componentization to the published Articles about Ulmar, Malvadine and Zhade

  • Added componentizable Content File, with their own Data Files and i18n entries, about the following Creatures’ variations: Ulmar, Wavescrambler; Ulmar, Perithon Racer; Malvadine, The King’s Herald; and Zhade, Dig Site Defender

  • Added componentizable Content Files, with its own Data Files and i18n entry about Ulmanator as a counterpart of the reference made in the recently componentized Article about Ulmar

Public Content-related changes

  • Added Akarakk Cavern as a keyword for the published Article about Blazier

  • Added Cast for Episodes #24-25, Season #01

  • Added a direct alias to Rao’Pa Sahkk Chimegrid, without being inside Rao’Pa Sahkk

  • Published Episode #24, Season #01 with Brazilian Portuguese audio

Removed

Public Content-related changes

  • With the revamp in the implementation of how the appearances of Creatures, Locations and, in the future, Mugics, Battlegears and Attacks, are listed as keywords for the META Tag (see below) all occurrences of Episode/Season names have been removed from all Content Files throughout the site

  • Removed Márcia Regina as the Brazilian Voice Actor of Krystella from Episode #05, Season #01, since neither she or Cláudia Carli (see below) are the right ones

Changed

Structure-related changes
  • Renamed the Brazilian Portuguese translation of Hive from Colméia to Colônia which is more comprehensive in the Danians context
Hugo Templates-related changes
  • Revamped the way Episodes in which a Creature appears in the Animation is added to the list of keywords used in the proper META Tag. Now, instead of defining manually Episode name under the FrontMatter Parameter keywords they’ll be read from the TV Appearances’ Data Files
Public Content-related changes

  • Rewritten part of the recently componentizable Article about Ulmar, Wavescrambler so it sounds more related to the Creature in Perim than to the TCG itself

  • Rewritten part of the recently componentizable Article about Ulmar, Perithon Racer so it sounds more related to the Creature in Perim than to the TCG itself -AND- to not mention an almost useless strategy involving this Card

  • Revised all keywords of all published Articles

  • With the renaming in the Hive expression mentioned, all occurrences of the old expression have been updated throughout the site (Brazilian Portuguese only)

  • Despite Rao’Pa Sahkk Chimegrid being now a componentizable Article under Rao’Pa Sahkk, its name should remain the same, and, therefore, that has been adjusted throughout the site (except on links)

  • Replaced the currently identified, yet flagged as not confirmed, Brazilian Voice Actor of Krystella Márcia Regina with Cláudia Carli, in both Content Files and i18n entry for MyAnimeList, since it’s notable her voice is the same of many others Hollywood characters, like Skyler White (Anna Gunn) in Breaking Bad

  • Updated Changelog Page

Fixed

Structure-related changes

  • Fixed, with a palliative solution, the issue that was preventing the Taxonomy Terms Page used as data-source for the Cards’ search Tool to be minified, thus resulting in a slight performance improvement when using the Tool

  • Fixed an issue with the i18n identifier for Mipedim Oasis, Aichlyys Fest added in the previous revision in which because it was lacking the -aichlyys-fest substring it was overwriting the value of the identifier for Mipedim Oasis itself (English only)

Hugo Templates-related changes
  • Fixed an issue in which Characters names with HTML tags — like CodeMaster Crellan’s in the Episode Battle Lessons — where being kept when building the OpenGraph META tags
Public Content-related changes

  • Fixed a misinformation in the published Article about Tartarek in which Malvadine was mentioned to be a Defender while only his The King’s Herald variant is

  • Fixed an issue with the Language Switcher of the placeholder Page about Ikkatosh, the Aich King which had the link to the Brazilian Portuguese version pointing to a non-existing Page

Image-related Changes

  • Added Big Picture for Ulmar, Perithon Racer

  • Added Big Picture for Theb-sarr as a counterpart of the update in the published Article about Malvadine, The King’s Herald

  • Replaced the Card Image of Ulmar, Perithon Racer with one compatible with the recently added Big Picture for its Page

Miscellaneous changes

  • Updated Sitemap XML for Battlegears, Creatures and Index

  • Updated minified CSS and JS
