Added

Structure-related changes

  • Added Content Files, Data Files and i18n entries for: Rainspears (Attack); Fanfare of the Vanishing (Mugic); Nexus Fuse and Wind Whip (Battlegears); Iflar, the Crown Prince; Marquis Darini; Na-inna and Siado (Creatures) as counterparts of the references made in the Article about the Prince Mudeenu published in this revision

  • With the expansion on Data Files for Videos model (see below), new JSON files have been added, one for each episode in both languages currently available

Hugo Templates-related changes

  • In order to reduce the number of calls to Hugo’s getJSON function and thus reducing the access to the filesystem, since Hugo still doesn’t have support for deep traverse with its index function, we’ve implemented in the Header Template a global Scratch with the contents of the JSON Data File specified in a Content File’s FrontMatter Parameter data (if any). However, because of a bizarre and undebuggable conflict, this Scratch is not available within the Card Shortcode we use to build the Cards’ Information Box from the Content Files nor in the Partial Templates this Shortcode calls as a form of delegation, fractionating the HTML. Because of this odd behaviour, the same routine, that works perfectly if isolated from the full theme, has been replicated in the Card Shortcode, except for the global Scratch creation as it’s not necessary in there

  • Wrapped Hugo’s errorf used in the Partial Templates of Creatures and Locations (TV) triggered when a Data File doesn’t have an appearance entry, conditioning it to a global Site Parameter (debug)

Public Content-related changes

  • Added Cast for Episode #27, Season #01

  • Added an i18n entry for Lisa Ortiz, English Voice Actor for Lulu as a complement for the Cast added to the Episode An Easy Win

  • Added an i18n entry for Renato Márcio, the (likely) Brazilian Voice Actor for Krekk as a complement for the Cast added to the Episode An Easy Win

  • Published the Episode #27, Season #01 with Brazilian Portuguese audio

  • Published Article about Prince Mudeenu

CSS-related changes

  • Added, experimentally, top/bottom margins rules for the new horizontal Advertisement Block in the Home Page

  • Added a small top-margin to the Navigation Bar Toggler as an attempt to centre it with the logo for mobile devices

  • Added a palliative solution for the alignment of the Server Selection message bar and Available Mirrors on Video Pages for Mozilla Firefox

  • Added top/bottom margins to the Bootstrap’s Grid Row used to display notes on the Video Pages differently for both Desktop and mobile versions

  • Added a rule targeting specifically the Mozilla Firefox to adjust the thicker bottom-border under the Like/Dislike icons on the Sharing Actions “clone”

  • Added some top/bottom padding to the flyout menus for the mobile version

Removed

Structure-related changes

  • With the change on Data Files structure (see below), almost all Data Files previously located directly under /data have been moved one level deeper accordingly to the available language, which, by all means, represents files removal and the inclusion of a new ones

  • Removed from the Partial Template of SEO for Videos an instruction creating an invalid OpenGraph tags (article:content_tier and uploadDate) for the Video Content Type

CSS-related changes
  • Removed unused, unnecessary and already commented rules

Changed

Structure-related changes
  • Changed the Data Files structure in the /data folder. Now, instead of having all localized Data Files together in the same categorized folder they’ll now be separated into different folders named accordingly to the Language Code. So, for example, what used to be /perim/attacks/allmageddon.en.json and /perim/attacks/allmageddon.pt.json now is /en/perim/attacks/allmageddon.json and /pt/perim/attacks/allmageddon.json
Hugo Templates-related changes

  • Changed the implementation of the Home Page Template and the Base Template to change the position of the right chevron and to improve the overall readability of the code

  • Changed the implementation of the Base Template to change the position of the right chevron

  • Changed the both Default List Template and List Template for Videos to reflect the changes in the Data Files paths

  • Changed the Perim Template to make use of the global Scratch created with Page’s Data File Content, if any

  • Changed the main Template of SEO to make use of the global Scratch created with Page’s Data File Content, if any

  • Changed the Partial Template of Image & Thumbnail to make use of the global Scratch created with Page’s Data File Content, if any

  • Changed the Partial Template of SEO for Videos to make use of the global Scratch created with Page’s Data File Content, if any

  • Changed the Partial Template for SEO of Schema.org’s Cast to make use of the global Scratch created with Page’s Data File Content, if any

  • Changed the Partial Template for SEO of Schema.org for Videos to make use of the global Scratch created with Page’s Data File Content, if any

  • Changed the Partial Template for Like/Dislike Icons to make use of the global Scratch created with Page’s Data File Content, if any

  • Changed the Single Page Template for Videos to make use of the global Scratch created with Page’s Data File Content, if any

  • Changed the main Template of SEO to make use of Hugo’s new reflection methods, thus removing some warning during compile/execution time \o/

  • Changed the Partial Templates of Creatures and Locations to make use of the Data File preload replica created in Cards Shortcode*

  • Adjusted the List Template for Videos to take into account that Episode Details now come from their own Data Files instead of a FrontMatter Parameter

  • Adjusted the Single Page Template for Videos to take into account that Episode Details now come from their own Data Files instead of a FrontMatter Parameter

Public Content-related changes

  • Expanded the usability of Data Files for Videos by moving all information about them to their own JSON. With this change all Content Files referring to a Video Episode has been updated

  • Updated Changelog Page

CSS-related changes

  • Adjusted the rules of the right chevron icon as a counterpart of the change in the Templates (see above)

  • Improved the Navigation Menu for mobile devices by adjusting the width of the links to their text length. This way the flyout menu opens only when clicking the text instead of opening when clicking somewhere to the right of them

  • Changed the colour of the link hidden in a flyout menu under the three dots in the Sharing Action Bar

  • Adjusted the margins of the Level 2 and 3 Headings (h2 and <h3>) as a counterpart of the change in the Templates (see above)

  • Adjusted the top-margin of the Advertisement Block on Video Pages

  • Redone Bootstrap’s padding definition for the Breadcrumbs Bar

  • Improved the spacing of Image Credits between Card Images and their respective Information Box

  • Improved the alignment of the little arrows indicating something will happen when hovering/clicking a link in the Navigation Menu

  • Adjusted the right positioning of the Languages’ Switcher to match the left distance of the Navigation Menu for mobile devices

  • Adjusted the top-margin of the Server Selection message bar, the Available Mirrors and the Video Player wrapper on Video Pages

  • Changed the design of the highlighted result’s details panel on the Home Page. Now, instead of appearing on the hover like the unhighlighted ones, it’s visible by default. And with an almost transparent top-border, it looks like the Big Picture is inside an envelope

  • Centralized the unhighlighted results’ details panel texts on the Home Page, improving the readability of long Pages’ Titles

  • Improved the top-margin of Locations’ Cards relative to their immediate Section title

  • Adjusted the top-margin of Flavour Texts (a.k.a “Quotes”) below Locations’ Cards as a counterpart of the adjust made above

  • Reduced the width of the right chevron between Pages’ Section and Titles for mobile devices

  • Adjusted the thicker grey bottom-border of the Sharing Actions Menu “clone”

  • Increased the width of paragraphs tags within Level 3 Section Headings (<h3>) for mobile devices

  • Increased the left-margin of Navigation Menu for mobile devices as an attempt to smoothen the “bumping” that seems to cut-off part of the text caused when the Navigation Bar is collapsing/expanding

  • Adjusted the colours associated with each Card Type darkening a little bit the borders added to the Cards Information’s “cells”

  • Improved the alignment of the Discipline Icons in Cards Information Box

Fixed

Hugo Templates-related changes

  • Fixed an issue in the Partial Template for Schema.org for Videos in which the Pages’ Title built was the full version generated by the Partial Template of Schema.org’s Title (i.e. including the | Chaotic Online expression) while it shouldn’t, only the raw Pages’ Title should be added in there

  • Fixed an issue in the Partial Template for Schema.org for Videos in which because of an unwanted conversion of single quotes to right and left apostrophes the condition we created in the previous revision to have site’s headline as description when a Summary is not available was not applying in all Pages with an apostrophe in the Title (i.e. M’arrillian’s Names)

  • Fixed a misconception in the HTML structure of Base Template that was creating N <article> tags within another external article tag on Listing Pages

CSS-related changes

  • Fixed an issue with Discipline Icons in Cards Information Box that was causing them to repeat themselves in some resolutions

Image-related changes

  • Added a Card Image for Prince Mudeenu and his variation Champion of Guard as a counterpart for the Article published in this revision

  • Added Big Pictures for: Fanfare of the Vanishing (Mugic); Nexus Fuse (Battlegear); Iflar, the Crown Prince; Marquis Darini and Siado (Creatures) as counterparts of the references made in the Article about the Prince Mudeenu published in this revision
