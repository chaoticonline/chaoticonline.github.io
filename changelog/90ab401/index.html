Chaotic Changelog rev. 90ab401

Changelog

  rev. 90ab401

 

Added

Structure-related changes

  • Implemented something to cover possible uncertainties regarding Voice Actors with another entry named verified along to the name of each Voice Actor identified listed in the cast FrontMatter Parameter. If this entry is defined for that Voice Actor -AND- set as false a little superscript number is added next to the Voice Actor name

  • Added a default value through a recently added i18n entry for cases in which no Voice Actor has been identified for a listed Character

  • Added the missing FrontMatter Parameters section and group in the Terms of Use Content File (Brazilian Portuguese only)

  • Added a Hugo’s Compiler setting to prevent the so-called Fast Render Mode to be used by default. Just a little something to speed-up the server initialization while developing

  • Added the FrontMatter Parameter show with the new format of Season and Episode in all applicable Content Files

  • Added i18n (internationalization) entries for the newly added Pages: Bierk, Zapetur, Kaal and The Pits

Public Content-related changes

  • Added missing keywords to the published Article about Lake Ken-I-Po

  • Added the attacking privilege for the published Article about the Riverlands, in particular regarding the Erak’tabb’s Foothold version (English only)

  • Published Article about the Lava Pond

  • Added missing keywords about Searing Symphony and Fluid Fugue in Episode #10, Season #03

  • Added an additional keyword about Khybon, the Renegade in Episode #13, Season #02

CSS-related changes
  • Added a dotted line while hovering one of the (non-blocked) available languages of the Languages Switcher

Removed

Structure-related changes
  • With the inclusion of the new FrontMatter Parameter show (see above), the entries season and episode (and its sub-entries) have been removed from the FrontMatter Parameter details
Hugo Templates-related changes
  • Removed a condition that was preventing the second level Section Heading (<h2>) to appear in the Terms of Use Page
Public Content-related changes
  • With the review of Terms of Use Page (see below), the old Disclaimer Page, which had not much content already, has been removed, along with its related i18n entries and CSS rules
CSS-related changes

  • Removed several rules from the Card Types Colours section that were made obsolete with the reviewed link styles

  • With the revision of links styles throughout the site (see below), several unnecessary rules have been removed

Changed

Structure-related changes

  • With the inclusion of the new FrontMatter Parameter show (see above), the JSON+LD used for Videos’ Structured Data had to been updated pointing to the new entries

  • Changed the way the GDPR Cookie is defined with a more commonly used way of defining global variables: adding it to the window Object

  • With the change above, the conditions in Google’s Analytics’, Google’s AdSense’s and Facebook’s SDK’s Template Files, as well as GDPR Consent check, became simpler, being the latter also an attempt of preventing the GDPR “banner” of appearing on mobile devices when the User has already accepted the Terms of Use once

  • Changed the name of the GDPR Cookie, forcing a new acceptance of the updated Terms of Use

  • With the removal of the Disclaimer Page, the footer has been updated to remove the link to it

Hugo Templates-related changes

  • Because of the addition of the verified entry (see above) the superscript 1 referring to the subtitles availability has been changed to 2, except when all Voice Actors are identified with certain, case in which no number 1 will exist

  • Replaced the palliative solution to house the subtitle availability message from an empty <dt> to an additional Bootstrap Grid’s Row. Not only it’s the correct approach but it’s also better to style with CSS

Public Content-related changes

  • Added M’arrillians as Enemies in the published Article about Takinom

  • Updated the comments regarding the FrontMatter Parameter cast in first episode’s Content File

  • Modified a paragraph in the following published Articles: Maxxor, Najarin, Najarin, Chaor, H’earring, Takinom, Glacier Plains, UnderWorld City and Riverlands to make even more evident that what was being said occurred in the Animation

  • The attacking privilege paragraph in the published Article about the Crystal Cave has been expanded, making clearer which Creature will attack and what’s the effect of the Location itself (English only)

  • Minor complement regarding the attacking privilege in the published Artigle about Lake Ken-I-Po, in particular regarding the Ritual Site version (English only)

  • Updated the Terms of Use Page with text revision, in order to include the Subsection about Cryptocurrency Mining, format revision to include a ToC Tree, and ortho-grammatical corrections

  • Updated Changelog Page

CSS-related changes

  • Balanced the top margin of all Heading Title tags, adjusting others as necessary — e.g. the one of second level (<h2>) had its bottom margin reduced in order to not make the one of third level (<h3>) become misaligned

  • With the change above, the margins of the horizontal line (<hr>) has been adjusted

  • Reduced a little bit the font-size of the fourth level Heading Title (<h4>)

  • Reduced even more the opacity of the hovering colour of the Social Links to the left

  • Balanced the bottom margin of Definition Lists (<dt>) as well as of Ordered (<ol>) and Unordered Lists (<ul>)

  • Increased the height of the Language Switcher’s little flags by 1px adjusting the ration with their width

  • Changed the colour of the emphasis tag (<em>) used on Terms of Use Page

  • Moved part of the Mipedians’ Colour Scheme more to the top because it ended up mixing itself up with the Tribeless’ Colour Scheme added in previous revisions

Fixed

Structure-related changes

  • Fixed the text entry in Pictures’ credits of several Content Files that wrongly remained in English in the Brazilian Portuguese version of the site

  • With the renaming of the i18n entries about Characters as part of the uniformization made in the last revision, one of the entries of the pubslidhed Article about H’earring was invalidated and has been fixed now

Public Content-related changes

  • Fixed a wrong keyword about the Lava Pond name in Episode #37, Season #01 which wasn’t properly translated (Brazilian Português only)

  • Fixed a wrong keyword about the Lava Pond, Milla’iin’s Foothold in Episode #18, Season #02 because there’s no evidence that the Location in which Maxxor battles Ragetrod is indeed the alternate version

  • Fixed wrong keywords about Khybon in Episode #16, Season #02 because, from that moment and on, he was under Milla’iin’s brainwashing and, therefore, the proper variant is Khybon, the Renegade

  • Fixed a wrong keyword in the published Article about the Riverlands, specifically regarding the Erak’tabb’s Foothold which wasn’t properly translated (Brazilian Português only)

  • Fixed a grammar issue in the published Article about Najarin (English only)

  • Fixed invalid local links (anchors) in the published Articles about Takinom, Maxxor and Chaor reminiscent after the name of the main version of the Creature had been removed from Section Titles — e.g: The Fierce instead of the old way Chaor — The Fierce — a few revisions ago

  • Fixed the Title of the Videos Page (/videos) that wasn’t appearing the Title Bar because its Content File didn’t have a Title defined as FrontMatter Parameter

CSS-related changes

  • Fixed the CSS selector for the wrongly generated empty <p> tags fix while parsing the Markdown

  • Fixed an issue with line-breaks (<br>) when used inside Quote Texts — e.g: Aa’une, the Oligarch — that, because of their styles, were causing lines to overlap on mobile devices

  • Fixed a weird issue in mobile devices in which after clicking the row of the first result in a Cards’ Search some semi-random results were appended to the table within the line clicked. The problem has been solved through CSS, by preventing a specific element to be clicked because, maybe, that’s a problem with the third-party plugin FooTable used

  • Fixed through CSS a weird issue, by preventing elements to be clicked, in which when the first row of a Card Search result is clicked some semi-random results were being created within the first line when viewed on mobile devices

Image-related Changes

  • Enhanced the Big Picture of the Lava Pond

  • Added Big Pictures for the newly added Pages: Bierk and The Pits

  • Added Card Images for Lava Pond and Lava Pond, Milla’iin’s Foothold for the Article published in this review

Miscellaneous changes

  • Updated Sitemap files (except Mugics)

  • Updated minified CSS and JS
