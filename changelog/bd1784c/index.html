Added

Structure-related changes

  • Added i18n entries to describe Locations in the OverWorld and UnderWorld in order to make it easier for the Templating logics to build i18n keys for them

  • Added Data Files to all existing Creatures published so far, with their attributes, traits, episodes and voice actors

  • With the Content Files Componentization applied to the published Articles about Locations (see below), new i18n entries have been added for each one of them

Hugo Templates-related changes

  • Added HTML attributes width and height to <img/> tags of “WIP” default images

  • Added a conditional route on Listing Template to consider cases in which the Data File entry picture of the Card being listed hasn’t even been defined in order to add the “WIP” picture instead of leaving its corresponding Bootstrap Card without an <img/> tag

  • Modified the markup of Navigation Bar to accommodate the new Site Logo \o/

Public Content-related changes

  • Added one more Breadcrumb Trail on Cards’ variants Pages pointing to their main versions to reinforce the relationship between them

  • Added missing Breadcrumb Trail Videos from Trivias Pages

  • Added a new Content File about Ulmar’s Lab

  • Added list of Episodes in which Vidav appears as well as his Voice Actor in both English and Brazilian Portuguese

  • Added missing weight FrontMatter Parameter to the published Article about Bodal, so it can appear at the offset it should (English only)

  • Added missing weight for the published Article about Lava Pond has been added, so it can appear at the offset it should (Brazilian Portuguese only)

  • Added missing keyword for Thonder in Episode #12, Season #02

  • While adding Data Files, Dractyl’s Big Picture credits has been added (Brazilian Portuguese only)

  • While adding Data Files, Heptadd’s Crown rarity has been added

  • While adding Data Files, Tharax’s Disciplines have been added based on data from Episode #12, Season #01

  • Published the Episode #19, Season #01 with Brazilian Portuguese audio

CSS-related changes
  • Added a few rules as counterpart of the new Site Logo \o/

Removed

Structure-related changes

  • Removed URL aliases pointing to anchor/local links from the published Articles about Bodal, Flamedrill Researcher and Takinom, the Shadowknight as they’re not that important

  • With the officialdom of the Content File Componentization feature (see below), the temporary Template Files used during the BETA period have been removed

  • Removed Content Files (and with them, the Pages) about Growlitzer (Battlegear) and Danian Hall (Location) since they’re not real Cards and have very little information to even consider becoming an Article

  • With the addition of Data Files for all Creatures and Locations, the corresponding FrontMatter Parameters have been removed from all Content Files

Public Content-related changes

  • Removed all placeholder paragraphs, of all Content Files

  • While adding Data Files, all Big Pictures credits referring to HotShots VFX had their links removed since their Facebook Page doesn’t exist anymore

CSS-related changes
  • Removed the negative margin fo unordered lists (<ul>) to prevent vertically aligned text to be come misaligned, especially with Flexbox

Changed

Structure-related changes

  • Officiated the Content File Componentization feature previously implemented as BETA feature only in the published Articles about Takinom and Bodal

  • Changed URLs of the componentized Articles about Bodal, Flamedrill Researcher and Takinom, the Shadowknight to become a sub-division of their main Cards. While this change might not seem much for Creatures with their unique names, this reduces the chances of conflicts for other Card Types

  • Renamed the Brazilian Portuguese version of Mipedim Oasis to Oásis de Mipedim, as it sounds more lexically correct

  • With the change above, the corresponding i18n entries have been adjusted as well

  • Modified the directory structure of Data Files for Creatures with variations to follow the same structure of /perim

Hugo Templates-related changes

  • Moved the logic of Advertising Positioning in Listing Templates all the way to the top to make it easier to read given the number of nested blocks

  • With the removal of FrontMatter Parameters in favour of Data Files, the Partial Templates for Meta Tags have been updated in order to reflect the changes made, therefore, returning the tags temporarily removed in the previous revision

Public Content-related changes

  • Applied Content File Componentization to all published Articles about Locations because they always used the Card Shortcode instead of Picture like Creatures and, therefore, with the officialdom of the feature previously implemented as BETA, they would stop working

  • Translated the physical traits height and weight of Bodal, Flamedrill Researcher to, respectively, meters and kilos in the Brazilian Portuguese version of the published componentized Article

  • Fixed all reminiscent occurrences of English’s possessive pronouns in the Brazilian Portuguese version of the site

  • Uniformed all measuring units to NOT have the period after the units’ abbreviation

  • Uniformed all Cards’ variations to have a line-break before their “subheading”, like their Cards

  • With the directory structure modification of Data Files (see above), the Include Shortcode of the published Pages about Takinom and Bodal, included within their main Articles, were updated accordingly

  • Updated Changelog Page

Fixed

Structure-related changes

  • Fixed Content File structure of the Article about Symphonic Shield missing the more special markup

  • Fixed several issues related to Structured Data JSON+LD created by a typo while implementing the new method for retrieving height and width of images

Public Content-related changes

  • Fixed letter-case of Whirling Wail (Attack) (English only)

  • Fixed list of Episodes in which Qwun and Antidaeon appear

  • Fixed Brazilian Portuguese Page about Cerbie whose link to the English version was wrongly pointing to Borth-Majar

  • While adding Data Files, Blazier’s physical traits skin and hair colours have been fixed (Brazilian Portuguese only)

  • While adding Data Files, Odu-Bathax’s Big Picture credits URL has been fixed as it was wrongly pointing to the same of Skartalas

  • While adding Data Files, Nauthilax’s physical trait height has been fixed (Brazilian Portuguese only)

  • While implementing the new FrontMatter Parameter data to associate a Content File with a Data File, the URL of the Page about Dractyl Scales (Battlegear) has been fixed (Brazilian Portuguese only)

  • Fixed a Markdown syntax for links used in the published Article about Najarin that weren’t properly interpreted and converted to HTML anchors

Image-related changes

  • Added a new image as Site Logo \o/

  • Updated Card Search Tool’s OpenGraph Image so now Facebook may display the text on it bigger as it should, like in the Articles/Videos

Miscellaneous changes

  • Updated Sitemap files (Index and Locations)
