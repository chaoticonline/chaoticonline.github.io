Chaotic Changelog rev. c228435

Changelog

  rev. c228435

 

Added

Structure-related changes

  • Added a custom sorting feature of the Characters listed in Episodes Cast through a new entry weight

  • Added new i18n entries with the Voice Actors of the following Characters: Sarah, Klay (by associating him with Chaor’s Voice Actor), Krystella, “HercuLeon”, “SlickNick”, CodeMaster Crellan, Tiaane (by associating him with Heptadd’s Voice Actor), Xaerv (by associating him with Magmon’s Voice Actor), Dractyl, Vidav, Agitos, Skithia and two possible Voice Actors (although one of them hasn’t been confirmed and, therefore, not used) to Owis (English only)

  • Added new i18n entries with the Voice Actors of the following Characters: Sarah, Peyton, Klay, Krystella, CodeMaster Crellan (real and “human form”), Zalic and Gespedan (by associating them with Zalic’s Voice Actor) and Xaerv (Brazilian Portuguese only)

Hugo Templates-related changes

  • Added the possibility of having HTML in Characters’ Name of Episodes Cast, so it’s possible to manually break a name that’s too long for the Column to hold it — e.g.: CodeMaster Crellan (Human Form)

  • Added a long dash (—) between the element identifying the Season and the Episode Number

  • Added a missing keyword with Kaz’s name ans screen name in Kaz Episode #20, Season #02

  • Added the following missing keywords to Episode #17, Season #01: Tharax, Dardemus, Malvadine, Mount Pillar e The Passage (UnderWorld)

  • Added Episode #17, Season #01 to Malvadine’s list of Appearances, since there’s already a published Article about him

  • Added Cast up to Episode #17, Season #01

  • Published the Episode #17, Season #01 with Brazilian Portuguese audio

CSS-related changes

  • Added a complement to the selectors of the Video Pages’ rules only as consistency to the other rules, since the elements styled there aren’t in use in the other Pages

  • Added a small top-margin to the blocks related to Videos Details and Observations (in case there’s any superscript number) so they get better aligned with the Cast block (if present)

  • Added Pagination Links to the list of selectors that won’t receive the dotted border when hovering the mouse

Removed

Structure-related changes
  • Removed, by commenting out, the URLs of the Human Characters in the respective i18n file so entries pointing to Pages that don’t exist (yet) won’t be added to the Structured Data. The feature remains the same for Creatures

Changed

Structure-related changes

  • Replaced the name Sarah Lawrence with Sarah “Beth” Cronwell in all Pages, mainly, but not restricted to, keywords because, after recent researches, this proved to be the correct name of her Character

  • Expanded all occurrences in all Video Pages’ Content Files of Kaz’s name to Kazdan “Kaz” Kalinkas Peyton’s to Peyton Touhey

  • Wit the expansion done with Kaz’s name, double quotes became part of the name string. Since this information is used mainly to build META Tags Structured Data in JSON+LD, they now reside within their own delimiters which are double quotes as well so, all occurrences, of all Template Files using them, they’ve been changed to escape HTML Entities

  • Changed the Cast structure allowing a better separation between Human and Creature Characters

  • With the change above, the Cast added to Episode #01, Season #01, previously added as a test, had all Characters’ names reordered

  • Updated the comments regarding the FrontMatter Parameter cast in first episode’s Content File

  • Changed the writing of the i18n entry related to subtitles availability

  • Changed the GDPR Cookie’s condition in the routine that updates the consent status (by showing/hiding the banner) to use the Cookie’s value directly instead of the global variable created in the <head>, making sure the banner would properly hide, which wasn’t happening before, in the first time a visitor who hadn’t agree/disagree with the Terms of Use yet. That was happening because the routine is executed as soon as the document is ready for manipulation but the global variable is created before that, so when it was trying to read the updated value of the Cookie, set when any of the buttons was clicked, the global variable had already been created with the default zero value as fallback if the Cookie didn’t exist yet. And this caused the GDPR banner to never hide, no matter how many time one of the buttons was clicked

Hugo Templates-related changes

  • Improved the disposition of the Cast in mobile devices by adding more of the of the Bootstrap Grid Classes

  • Even though Facebook Analytics isn’t being used anymore, its Template File still exist and the GDPR Cookie condition in it has been simplified

  • Changed Videos Pages’ Section Titles so the Cast is now of fourth-level (<h4>) and Video Details of fifth-level (<h5>), unless there’s no Cast defined, case in which it’ll remain as fourth-level

  • Changed the whole process used to build the Properties actor and character of TVEpisode Structured Data. Now we first read everything available, building an iterable data-structure of our own and then we iterate later over that much cleaner structure to build the Properties. This way we remove many apparently unnecessary conditions and hard-to-read nested blocks

Public Content-related changes
  • Updated Changelog Page
CSS-related changes

  • With the change in Videos Pages Section Title (see above), the tag which had its font size globally updated in previous revisions (h4) has been changed to h5

  • With the possibility of having HTML in Characters’ Names for better disposition of long names, some adjustments had to be made in the HTML markup and this caused the superscript number indicating uncertainty regarding a Voice Actor to be too far from the text. This has been changed by nullifying a Bootstrap’s default value

  • Because of the same reason described above, a little adjustment had to be made with the dotted border added when hovering links so other elements around the anchor don’t “jump”

  • And since we were updating the <sup> tag globally we took he time to reduce its font size a little bit

  • Improved the disposition of the Cast in mobile devices, as counterpart of the changes made in the HTML markup

Fixed

Structure-related changes

  • Fixed an issue that was preventing Voice Actors’ Profile URL list in the Cast to be found when an accentuated vowel was present in the first character of their name/surname since those are usually capitalized and we replace only the lowercase versions of them to simplify the process

  • Fixed the reason of Pages pointing to /videos/season-X still being generated when they shouldn’t since the big restructuring a few revisions ago the URL of each Season has been replaced with their respective names Until this fix, these Pages weren’t publicly available because their files/directories were being manually ignored during the commit

  • With the fix above the palliative solution that removed them with .gitignore has been removed

  • During the restructuring, the Episodes Section was renamed to Videos but, by accessing the respective link from Breadcrumbs Trails, that did not point to the expected Page in the same language browsed. This problem, caused by a minor typo i one of the i18n entries with the list of URLs available for template translations, has been fixed (Brazilian Portuguese only)

Hugo Templates-related changes

  • Fixed a logical issue in Google AdSense’s Template File that was preventing all advertising to appear. Prior to this fix, it was tested if the GDPR Cookie did not exist and if it didn’t exist it was being tested if its value was zero (?), meaning the non-consent with the Terms of Use. Because this obviously would never result in a positive test, the condition always failed and when that happened Google AdSense’s mechanism was explicitly configured to not show any advertising

  • Fixed an issue that prevented the correct functionality of Social Links, opening the Social Network chosen in popup for the Video and Article Pages

Public Content-related changes

  • Fixed double quote characters used for HTML escaping that remained visible in some of the Images Credits throughout the site because this information didn’t have any double quote delimiter, which is the precise reason of having escaping characters (Brazilian Portuguese only)

  • Fixed the Upload Date of the YouTube mirror of Episode #16, Season #01 with Brazilian Portuguese audio

  • Fixed a ma-formation with the FrontMatter Parameter mirror in the Episodes #18, #19, #20, #22, #29, #31, #33, #35 and #38, all from Season #01 (Brazilian Portuguese only)

Image-related Changes

  • Added a Big Picture for the unpublished Article about Zapetur

  • Changed the Big Picture for the unpublished Article about Quadore

Miscellaneous changes

  • Updated minified CSS and JS
