Added

Structure-related changes
  • Added Content Files, Data Files and i18n entries for the following Pages as counterpart of the references made in the Article about Magmon published in this revision: Fireram (Attack), Whepcrack (Battlegear) and Kreaal; Magmon; Retaliator and Magmon, Engulfed (Creatures);
Hugo Templates-related changes

  • Added an empty element to the bottom of the Card Information Box with the same colour of the Card Type / Tribe, visually closing the box

  • Added a condition to the Card Information Template to -NOT- show the Biography Header if there’s nothing to show under it

Public Content-related changes

  • Added missing FrontMatter Parameter weight to the published Article about Wamma to position it in the proper offset accordingly to its appearance in the Animation (even though he’s not the first Danian to appear)

  • Added missing instruction to the Data File of the main version of Wamma which caused the first Information Box to not render

  • Added an instruction to the Data File of unpublished Article Rath’tab to show the custom Card made for him

  • Published the Episode #20, Season #01 with Brazilian Portuguese audio

CSS-related changes

  • Added all rules needed to build a colour scheme for M’arrillians’ and Tribeless’ Creatures

  • Added all rules needed as counterpart of the new Tribeless Creatures Elemental Affinity Icons added to the UI Sprites image file

  • With the horizontal scroll fix (see below), because the rules added to squeeze more visible space have been removed, the left/right columns of the footer needed some margins to not stay too close of the screen borders

Removed

Public Content-related changes

  • Removed the Card Shortcode attribute source form the published Articles about Takinom and Bodal as they’re not needed anymore

  • Removed a duplicated special markup more from the published Article about Wamma that was causing content to appear before the Flavour Text (Brazilian Portuguese only)

CSS-related changes

  • Removed some duplicated rules in the Card Type / Tribe colouring section of the CSS file

  • With the horizontal scroll fix, a few other rules previously added to squeeze more visible space have been removed because they were causing a negative impact

Changed

Hugo Templates-related changes

  • Modified the Card Information Template to show the Information Box regardless the existence of the custom-made Card in the defined path. If the Card doesn’t exist, a default Card inspired in the back side of the original Chaotic Cards will be added as a placeholder

  • Adjusted the Bootstrap Grid classes for Creatures so those with Super Rare and Ultra Rare Rarity won’t break into two lines in English and languages with similar word lengths

  • Improved the horizontal alignment of the Card Images relatively to their the Information Box

Public Content-related changes

  • Improved the positioning of the Big Picture of Carnival of Confusion on big viewports

  • The paragraph under the The Animation section of the published Article about Lava Pond has been updated in order to be more on par with the writings of the Article about Magmon published in this revision

  • Updated Changelog Page

CSS-related changes

  • With the inclusion of the new Elemental Affinity Icons for Tribeless Creatures, the overall format of the CSS Sprites image file changed, with some icons repositioned and, therefore, all background-position and some height and width rules have been updated

  • Improved the horizontal alignment of the Big Pictures, mostly noticeable on smaller viewports

  • With the fix with Location Cards dimensions (see below) -AND- with the improvements made to the Card Information Template (see above), their Image Credits’ texts had to be slightly adjusted as well

  • Increased significantly the dimensions of Locations Cards

Fixed

  • Fixed an issue with the i18n entry about Mount Pillar Reservoir in which, because its identification key lacked the expression -reservoir, its value was overwriting the entry immediately before about Mount Pillar (Brazilian Portuguese only)
Public Content-related changes
  • Fixed a misinformation in the componentizable published Article about Najarin, Younger since three Creatures with the same name, regardless of their variant, cannot coexist in the same squad
CSS-related changes

  • Fixed a horizontal scroll on mobile version caused by the our reduction on Bootstrap’s default margins at the cost of having slightly larger margins around the main content, thus reducing the visible area

  • Fixed an issue with Location Cards getting super small on mobile because of the new dimensioning units implemented in the previous revision

  • Fixed an issue with the colouring of Quotes Authors’ Links (Flavour Texts) of Locations Pages (and, if not fixed, M’arrillian’s and Tribeless’ as well) in which, because of a wrongly defined declaration for Links and Italic-styles for Danians, everything else below it would get the brownish colour defined for them (Danians), instead of their own colours (respectively, green; dark bluish-green; and grey)

Image-related changes

  • Added placeholder Cards, in both landscape and portrait orientations, inspired in the back side of the original Chaotic Cards as counterpart of the change made to the Template Files for Creatures and Locations

  • Added a Card Image for Rath’tab, mostly as a demo of the M’arrillian Card Template

  • Added a Card Image for Ursis, mostly as a demo of the Tribeless Card Template created

  • Because the Card Template for Tribeless Creatures didn’t exist before, there were no Creatures’ Elemental Affinity Icons with their colour scheme (greyish). Now that the Card Template does exist, these icons became available and thus have been added to the remodelled UI CSS Sprites image file

  • While restructuring the CSS Sprites image file, the logo of the Video Providers have been removed from the Cards’ Icons file to be grouped in one of their own as they are never used together, which will slightly improve the loading time of Episodes’ Pages

  • Added the Big Picture of Whepcrack for the new Content Files added as reference for the Article about Magmon published in this revision

  • Updated the following Big Pictures with versions of better quality: Allmageddon, Hail Storm, Lavalanche and Megaroar (Attacks), Aquashield, Cyclance, Flux Bauble, Ice Cloak, Liquilizer, Mowercycle, Pyroblaster, Ring of Na’arin, Riverland Star, Skeleton Steed and, as a special partially redone work, the Stone Mail (Battlegears)

  • Updated ALL pre-existing Creature Cards to align the Creature Type text (Hero, Warrior, Scout…) with the Card Description -AND- to publish them as Optimized JPEG files, instead of Progressive

  • Updated ALL pre-existing Location Cards to increase the height of the green band with the Card Type (fixed), to better align any icons among their Descriptions -AND- to publish them as Optimized JPEG files, instead of Progressive

Miscellaneous changes

  • Created a new Card Template for Tribeless Creatures

  • Updated Sitemap files for Attacks, Battlegears, Creatures and Index

  • Updated minified CSS
