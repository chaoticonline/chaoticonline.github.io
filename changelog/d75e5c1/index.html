Chaotic Changelog rev. d75e5c1

Changelog

  rev. d75e5c1

 

Added

Structure-related changes

  • Added componentizable Content File, with their own Data Files and i18n entries, about the following Creatures’ variations: Blazier, Citadel Cartographer; Tartarek, Psi-Overloader; Chaor, The Fierce; H’earring, Tainted; Nauthilax, Tainted; and Nauthilax, Aquatic Spy

  • Added Content Files, Data Files and i18n entries for Reefshard (Battlegear) as a counterpart of the reference made in the recently componentized Article about Tartarek, Psi-Overloader

  • Added Content Files, Data Files and i18n entries for Decrescendo (Mugic) because, well, the artwork was too beautiful to pass :p

  • Added an i18n entry for César Marchetti as the second Brazilian Portuguese Voice Actor of Staluk

  • Added Data Files for the identified voices of Ettala, Gespedan Mommark, Staluk and Klasp (Brazilian Portuguese only) and Owis (all Languages)

  • With the recently identified voices, in order to show the “On TV” box, the following Data Files under TV Appearance have been added: Ettala, Gespedan, Mommark, Owis, Staluk and Klasp

  • Added Content Files, Data Files and i18n entries for Mommark and Klasp as a counterpart of the recently identified Voice Actors

Public Content-related changes

  • Added missing Flavour Text to the Pages about Dread Tread and Obliterators, added as references to the Article about Dractyl published in last revision

  • Added missing keyword in the published Article about H’earring as a reference to the Episode Newbie, which marks the beginning of his brainwashed state

  • Added Cast to Episode #23, Season #01

  • Published the Episode #23, Season #01 with Brazilian Portuguese audio

CSS-related changes
  • Defined a fixed a max-height for Big Pictures on Desktop versions

Removed

Structure-related changes
  • Since the revamped SEO implementation is working consistently as expected, all Partial Templates under /partials/meta have been removed
Public Content-related changes

  • Removed Light Purplish-grey as one of the skin colours of Najarin, Younger as this is just in the Big Picture we’ve found — which is the image used in the Card Art re-creation — while the original Card Art his skin is Light Azure-blue as it always were

  • With the Content File Componentization applied to the main published Article about Tartarek, information belonging to his Psi-Overloader variation, namely, colours schema and loyalty to the M’arrillians, have been removed from his Data File

  • With the Content File Componentization applied to the main published Article about H’earring, information belonging to his Tainted variation, namely, colours schema and loyalty to the M’arrillians, have been removed from his Data File

  • With the Content File Componentization applied to the main published Article about Nauthilax, information belonging to his Tainted variation, namely, colours schema and loyalty to the M’arrillians, have been removed from his Data File

  • Removed M’arrillians as enemies of H’earring from his Data File, after all, the first appearance of H’earring didn’t have any contact with the M’arrillians yet

Changed

Structure-related changes

  • As we’re moving our development Environment from Sublime Text to Visual Studio Code, the gitignore file has been updated to ignore something else

  • Applied Content File Componentization to the following published Articles: Blazier, Tartarek, Chaor, H’earring and Nauthilax

Hugo Templates-related changes
  • Improved Creatures and Locations Partial Templates to simplify the markup of Image Credits by always providing them in the structural Hugo’s Scratch
Public Content-related changes

  • Rewritten part of the recently componentizable Articles about H’earring, Tainted; and Nauthilax, Tainted so they sound more related to Creatures in Perim than to the TCG itself

  • Rewritten part of the componentizable published Article about Takinom, the Shadowknight

  • With the recently identified voices, in order to show the “On TV” box, the following Pages have been updated: Ettala, Gespedan, Owis and Staluk

  • Updated Changelog Page

CSS-related changes

  • Reverted the changes to Information Box top/bottom margins because they were messing with the Flavour Texts element below (if any)

  • Reduced a little bit the bottom margin of the Flavour Text element of mobile version

  • Made a micro-adjust in the vertical positioning of Image Credits between the Card Image and the Information Box on the mobile version

  • Reduced the main left/right margins on the mobile version greatly improving the overall disposition and readability of smaller elements, like the Information Box

  • Improved the vertical and horizontal alignment of the Discipline Icons and their corresponding values with some micro-adjustments

  • Reverted the negative bottom-margin of Fallback Card Images made in previous revisions because while they look a bit too much in Articles without any content at all, in those with the bare minimum structure after voice identification (i.e. Section “In Animation”) it becomes all messed up

Fixed

Hugo Templates-related changes
  • Fixed an issue with Breadcrumbs getting duplicated N-times as far as the Listing Section Pages increase
Public Content-related changes

  • Fixed the Data File associated with the published componentizable Article about Bodal, Flamedrill Researcher, which, because of a typo in the JSON structure his Image Credits weren’t appearing

  • While creating the Data Files needed to apply the Content File to the published Article about Nauthilax, his Gender has been fixed as it wasn’t translated to the proper locale (English only)

  • Fixed an issue with Cards’ Search Tool in which because of the lack of quotes around the href attribute in the links of Taxonomy Terms Page (/tags) all asynchronous Requests were failing, considered as Mixed Content. The solution, paliative, is by manually not minify the said Page

Image-related changes

  • Replaced the Big Picture of Ash Torrent (Attack) with a version of better quality

  • Added Big Pictures for the following componentized Articles recently added: Tartarek, Psi-Overloader; Chaor, The Fierce; H’earring, Tainted; and Nauthilax, Tainted

  • Added Big Picture for the recently added Page about Klasp (Creature)

  • Added Big Picture for the recently added Page about Decrescendo (Mugic)

Miscellaneous changes

  • Updated Sitemap files for Mugics, Battlegears, Creatures and Index

  • Updated minified CSS and JS
