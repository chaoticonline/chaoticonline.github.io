Added

Structure-related changes

  • Added an external plugin to handle the Clipboard, allowing texts to be copied to it

  • Added Bootstrap Modal Component as part of the new method of Sharing Article/Pages

  • Added Content Files _index.<LANG>.md under /changelog in order to enable it as a Hugo Section as part of the new implementation of Changelog Pages (see below)

  • Added a new possibility for the FrontMatter Parameter advertisable named sidebar, allowing Pages to show advertisement blocks only in the sidebar

  • Added a new Data File for the revamped Home Page with, mostly, the path of the Chaotic Logo (see below)

  • Added a new Data File with a structure to translate Weekdays and Months as a compliment for the implementation of Changelog Pages (see below)

  • Added missing i18n entries for the Componentizable published Article about Tangath Toborn, OverWorld General

  • Added i18n entries for the new Social Networks as a compliment for the new method of Sharing Articles/Pages (see below)

  • Added new i18n entries for the Sections and Titles of the revamped Home Page (see below)

  • Added a new SVG Icon for the collapsible menu toggler as the new version of Bootstrap seems to have dropped support for the inverse background (dark grey)

  • Added a new icon for the new Sharing menu as well as for all the new Social Networks as a compliment for the new method of Sharing Articles/Pages (see below)

  • Added a new SVG Icon to indicate that a link will open in a new tab/window as complement for the revamped Home Page (see below)

  • Added missing SVG viewBox value for Twitter’s symbol

Hugo Templates-related changes

  • Implemented a new method of Sharing an Article/Page with more Social Networks available

  • Implemented a way to like/dislike an Article/Page

  • Implemented a Card Preview in the Cards’ Search Tool, directly from each result found (Creatures and Locations only)

  • Implemented a better way of handling Changelog’s Revisions. Now instead of having all of them in one single Page, relying upon a JS workaround to collapse big Revisions, now it acts like a regular Results’ Listing, decreasingly ordered and based on the special markup MORE, showing the first changes of each and providing a link to the full Page

  • Added an alternative Paginator definition for the Changelog Page

  • Added, through Bootstrap’s Utility Classes, as small top-margin to the Cards’ Information Box

Public Content-related changes

  • Added a missing FrontMatter Parameter publishingDate in the published Articles about Dractyl and Kiru City

  • Added missing FrontMatter Parameter initiative to be used as Taxonomy Term in all Locations’ Content Files without it

  • Added the possibility to filter out some specific Pages from Cards’ Search Tool’s Results. Currently, only Rao’Pa Sahkk is ignored in this way because it’s a big Location made of smaller sub-locations and only the smaller ones are real Locations

Removed

Structure-related changes

  • Removed the external CSS file readmore.css and the external JS files readmore.js, abbr-touch.js and touchtap-event.js as they’re not needed anymore (see below). This, in turn, will provide a small performance boost in Pages’ loading times

  • Removed the Content Files changelog.<LANG>.md as it’s not needed anymore as part of the new implementation of Changelog Pages (see below)

Hugo Templates-related changes

  • Removed the attempt of loading the compiled CSS and JS files without SSL while running through Hugo Server because that ended up not working as exepected when tunnelling

  • With the new method of Sharing Articles/Pages, the <aside> where the old Social Network icons used to be has been removed. This, in turn, gave more room for the main content

  • Removed the code-block about Dates (created, published and modified) from the SEO Partial for Listing Pages because the same code, now improved even further, was already present in the main Template

  • Removed the alternative route for when a Page has no description (summary) because from now, one way or another, they all will

  • Removed Bootstrap’s Utility Class to fix a full maximum width (w-100) from Google AdSense’s wrapper block in the Videos Partial Template

Changed

Structure-related changes

  • Updated Bootstrap version from 4.0.0-alpha.6 to v4.2.1

  • Changed the way the expanded <abbr> definition is displayed on mobile devices and, with this change, the feature also became available on Desktop

  • Changed the IDs of the CSS Ribbons <_<

  • Revamped the Home Page. Now instead of just a big, centred logo, with the headline below, now the Home Page lists the last published Articles and uploaded last Videos, with thumbnails and “hoverlays” with Pages’ Title and, for the most recent of each category, highlighted with a bigger thumbnail, Page’s Summary as well

Hugo Templates-related changes

  • With the updated in Bootstrap, the old way of hiding content accordingly to the viewport changed and, therefore, several occurrences had to be updated throughout the theme

  • Updated the Partial Template Base to include the necessary markups for the new feature of Like/Dislike and Sharing of an Article/Page, as well as to replace generic <div> with the proper <header> and <article> tags

  • Revamped the implementation of the Results Listing Template restructuring the Bootstrap’s Cards markup, which in turn improved the Page responsiveness

  • With the new implementation of Changelog’s Revisions, the format of the Committing Date was a little expanded to also include the translatable weekday

  • Returned the Fingerprinting feature of compiled and minified files

  • Simplified the top navigation bar markup, both the wrapper and the menus

  • Updated the main Partial Template for SEO in order to:

    • Make data about Big Picture and Page’s Description global, this way it can be used outside the SEO Partials, specifically as part of the new method of Sharing Articles/Pages
    • Grant to all Pages, even those missing the proper FrontMatter Parameter, values for the Schema.org’s Properties dateCreated and datePublished
    • Have the headline as Page’s Description when the Page has none (no Summary), which is the case of the Home Page and Articles without real content
    • Invert the condition that includes the theme-specific Partials (namely Articles or Videos) in order to be able to filter out the Home Page since it isn’t a Hugo’s Regular Page and, because these Partials use resources available exclusively for these kinds of Pages, instead of simply raise an error, the server/compiler simply panics and aborts everything o.O

  • With the change regarding the data scope of the Big Picture and Page’s Description, all occurrences throughout the site have been updated

  • Updated the SEO Partial for Images to provide the Chaotic Logo as an image if no custom Data File exist for that page -OR- if the path provided on them doesn’t resolve to an existing image. This way all Pages now have an Image to create the proper Schema.org’s Entity

  • Simplified the Card Picture markup in the Partial Template for Location Cards

  • Drastically improved the responsiveness of Locations Pages

  • Simplified the markup of the Shortcode Picture, currently in use only the published Article about Crystal Cave to show the picture of the Akkarak Cavern

  • Simplified the condition in the Partial Template for Results Listing for Videos that distinguishes Regular Episodes from the Trivia

  • Simplified the way Content Files have their.. content added to the HTML, removing several workarounds to Hugo’s rendering inconsistencies

Public Content-related changes

  • Minor update in Cards’ Search Tool Page “funny” text

  • With the change in the implementation of Changelog Page (see above) all Changelogs’ Content Files have been updated to reflect the changes, modifying their FrontMatter Parameter category and adding a special markup MORE along the text

  • Reverted Google AdSense’s Advertisement Block Format from horizontal back to auto from the following published Articles: Tangath Toborn, Malvadine, Wamma, Takinom, Najarin Glacier Plains and Iron Pillar. The published Article about Najarin, in particular, has been chosen to test a new, recently created, Advertisement Block format/layout. After the testing period, the most suitable one will remain in all Pages

  • Updated the Content Files of all unpublished Articles to manually include the Page Title right above the special markup MORE as a counterpart of the improvements in SEO Templates (see above)

  • Updated the Content Files of all published Articles changing the Section Title Game & Combat to a more generic Combat, allowing, in a future update, their content to be more related to Creatures themselves rather the Card in the Game

  • With the (unnecessary) change in the CSS Ribbons identifiers, all Content Files _index.<LANG>.md under /videos have been updated to reflect the changes

  • Changed the CSS Riboon ID of Season #01 from Gray (now named Ash) to Dark Red (now named Blood)

  • Revised all Video Upload Date/Time accordingly to the associated Stream Platforms and all Publishing Dates accordingly to the Facebook Posts

  • As a counterpart of the simplification made aiming rendering inconsistences minimization, all Content Files have been revised regarding the positioning of their special markup MORE and number of paragraphs and/or blank lines

  • Marked the main page of Rao’Pa Sahkk as hidden by setting its FrontMatter Parameter listed to false, after all, only its sub-locations are real Locations

  • Updated Changelog Page

CSS-related changes
  • File updated with all rules necessary for the new resources as well with several minor adjustments

Fixed

Hugo Templates-related changes

  • Fixed an awkward apparent incompatibility between Hugo and its Markdown parser “Blackfriday” in the SEO Partial for Titles that was yielding different results for strings with apostrophes

  • Fixed an issue that could prevent default Image Credits values in the Partial Templates of Creatures and Locations

Public Content-related changes

  • Fixed the Brazilian Portuguese URL of the unpublished Article about Cyclance

  • Fixed the Brazilian Portuguese Title of the unpublished Article about Burithean Axe

  • Fixed the Brazilian Portuguese Title of the unpublished Article about Raznus, Assimilated

Miscellaneous changes

  • Turned off ESLint’s configuration regarding the obligation of having a default clause in a switch block

  • Added in the config.yaml file a hardcoded value with Site’s Creation Date, mostly to be used along the revamped Home Page

  • Added in the config.yaml custom Parameter debug through which custom debug with Hugo’s errorf function will be triggered

  • Updated minified CSS and JS
