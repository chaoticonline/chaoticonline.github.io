Chaotic Changelog rev. 7b09115

Changelog

  rev. 7b09115

 

Added

Structure-related changes

  • Added default values for the META Tags description if a Summary, the text before Hugo’s special markup <!--more-->, hasn’t been defined -AND- if after replacing all HTML line-breaks (<br/>) with new-lines (\n) the resulting Summary is an empty string or with only non-printable characters. This way we ensure that all Pages, even those not effectively published yet due lack of real content has a value for the tag

  • Added Bing’s Authorization XML file so it can properly crawl the site because, apparently, Microsoft doesn’t know understand that, as of today, quotes around the HTML attributes of these tags and/or ending slash for self-closed tags are optional

Public Content-related changes

  • Added missing FrontMatter Parameter publishingDate to the published Articles about Magmon (Creature) and Lava Pond and Iron Pillar (Locations) and on each one of the componentizable revisions in the Changelog Page, even though such information will not be actually added on them while the revamped SEO implementation is not applied to the Videos’ Pages because of FrontMatter Parameter conflicts between them

  • Added missing FrontMatter Parameter group to the following published componentizable Articles: Bodal, Flamedrill Researcher; Maxxor, Protector of Perim; Najarin, High Muge of the Lake; Najarin, Younger; Takinom, the Shadowknight; Wamma, Flame Guard; and Wamma, Hive Ordnance

  • Added missing FrontMatter Parameter group to the published componentizable Article about Najarin, Fluidmorphers’ Foe (Brazilian Portuguese only)

  • Added missing FrontMatter Parameters publishingDate -AND- group to the following published componentizable Articles: Glacier Plains, M’arrillian Melting Camp; Glacier Plains, M’arrillian Heat Cannon; Lake Ken-I-Po, Ritual Site; Riverlands, Abandoned Wastes; Riverlands, Erak’tabb’s Foothold; UnderWorld City, During Van Bloot Ascension; and Lava Pond, Milla’iin’s Foothold (Locations) and Magmon, Retaliator; and Magmon, Engulfed (Creatures)

Removed

Hugo Templates-related changes
  • With the inclusion of Bing’s Authorization XML file (see above), its corresponding <meta> tag has been removed from the Header Partial template

Changed

Hugo Templates-related changes
  • Changed the FrontMatter Parameter publishingDate of ALL published Articles/Pages as a counterpart to the fix of format passed on to Hugo’s dateFormat function (see below)
Public Content-related changes
  • Updated Changelog Page

Fixed

Structure-related changes
  • Fixed robots.txt with recommendations of Yandex Webmaster
Hugo Templates-related changes
  • Fixed the format passed to Hugo’s dateFormat function in order to take into account timezones as well. For future references, the full format follows RFC 3339 and can be found in the Go’s docs

Miscellaneous changes

  • Updated all but Videos Sitemap Files with an extended <urlset> tag definition with recommendations of Yandex Webmaster
