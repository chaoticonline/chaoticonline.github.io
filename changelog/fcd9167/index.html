Chaotic Changelog rev. fcd9167

Changelog

  rev. fcd9167

 

Added

Structure-related changes

  • Implemented Content File Componentization in the published Article about Takinom and Bodal as BETA feature using the improved Include Shortcode, putting the subsection about Takinom, The Shadowknight and Bodal, Flamedrill Researcher on its own Page. This way not only main Articles will be shorter after the live testing period but all Cards’ variations will appear in a Cards Search results because every one of them will have their own URL

  • Added an optional FrontMatter Parameter listed through which it’s possible to remove an Article/Page from Listings without remove it from the site

  • Added Content Files about Takinom, the Shadowknight and Plasmarrow, as counterpart of the many changes made in the published Article about Takinom

  • Added Content Files and i18n entries for Takinom, the Shadowknight and Plasmarrow as, respectively, part of the Content File Componentization and counterpart of one of the changes in Article about Takinom

  • Added i18n entries for Disciplines, Elemental Affinities and some others more specific in order to have matches for Taxonomy Terms

  • Added the following Content Files and i18n entries referenced in the Article about Bodal published in this revision: Lucky Shot, Supercooled Rain and Viperlash (Attacks), Mount Pillar Reservoir, Gloomuck Swamp and Kiru City Tunnels (Locations), Bodal’s Dagger, Orchis Undis and Sandstorm Shroud (Battlegears) and Olkiex and Bodal, Flamedrill Researcher (Creatures)

Hugo Templates-related changes
  • Returned the Ribbons with Episodes’ numbers in their Thumbnails that was accidentally removed when the FrontMatter Parameters Structure changed a few revisions ago
Public Content-related changes

  • Added missing keyword for Bodal in Episode #08, Season #02

  • Added list of Episodes in which Yokkis appeared as well as his Voice Actors in both English and Brazilian Portuguese versions of the site

  • Complemented the section Game in the Article about the Glacier Plains referencing the wrongly depicted effects of it seen during the events of Episode #18, Season #01 (Allmageddon)

  • Added missing Shortcode markup to allow Cards Search Tool to read the rarity value from the Articles about Mezzmarr, Tangath Toborn and Wamma

  • Added missing Shortcode markup to allow Cards Search Tool to read the rarity and initiative of (almost) all Locations

  • Added missing Taxonomy Term type for Song of Futuresight (Brazilian Portuguese only)

  • Published Article about Bodal

Removed

Public Content-related changes

  • Removed the FrontMatter Parameters height and width from a few Content Files as these values come automatically from now on (see below)

  • Removed all occurrences of attribute class from Cards Shortcode, since it has no effect anymore, from the following published Articles: Maxxor, Najarin, Blazier, Heptadd, Tartarek, Chaor, H’earring, Nauthilax, Takinom, Ulmar, Malvadine, Zhade and Lake Ken-I-Po

  • Even though the Content File Componentization has been implemented as BETA feature in the publichsed Article about Takinom (and Bodal), all FrontMatter Parameters’ comments related to the removed Parameters have been removed

  • Removed one of the Deferred Callbacks from the main Cards Search Tool script exclusively to show the Loading Icon after resetting all UI styles as the benefits offered by this approach were minimal

Changed

Structure-related changes

  • Implemented a feature that’ll allow Creatures’ physical traits from FrontMatter Parameters to Data Files, allowing better organization and readability but, most important, the possibility of having Cards’ Variations’ Details present structurally in the markup, allowing them to be indexed by Cards Search Tool. For now, it’s only implemented in the Articles about Takinom and Bodal as a BETA feature

  • Implemented a feature that’ll allow the list of Episodes in which a Creature/Location appears from FrontMatter Parameters to Data Files, allowing better organization and readability but, most important, auto-linking. For now, it’s only implemented in the Article about Takinom and Bodal as a BETA feature

  • Implemented a feature that’ll allow identified Voice Actors of at least one of the languages of the site from FrontMatter Parameters to Data Files, allowing better organization and readability. For now, it’s only implemented in the Article about Takinom and Bodal as a BETA feature

Hugo Templates-related changes

  • Changed the way HTML attributes height and width are added to the <img/> tags. Now instead of reading a FrontMatter Parameter, Hugo’s Image Metadata Reader function imageConfig is used instead

  • Likewise the change above, the height and width values defined in JSON+LD Structured Data and <meta/> tags were changed using the same approach described above

  • Improved the Include Shortcode that was halted due hard times to find specific information on how to create componentizable Content Files with Hugo

Public Content-related changes

  • Typo, grammar and wording corrections and additions for the following Articles/Pages: Maxxor, Heptadd, Tartarek, H’earring, Nauthilax and Takinom

  • Renamed the HTML markup of Cards Search Tool to use the Parser name for All Cards as value of the Card Type dropdown

  • With the change above, the Tool’s main script, Search.js, has been adjusted to only instantiate the parser for All Cards when and if needed, instead of instantiating it by default and then overwrite it with a different one, thus improving performance

  • Updated Cards Search Tool to take into account the Content File Componentization implemented as a BETA feature in the Articles about Takinom and Bodal

  • Improved the Cards Search Tool so now it doesn’t need a “Swap Area” to alphabetically order the results by anymore

  • Improved the Cards Search Tool main script to not look for a specific selector multiple times by caching it, thus slightly improving performance

  • Improved the Cards Search Tool Filter.js script to not require Filter::filter() to be manually invoked in the main script by wrapping both actions, intersecting and retrieving filtered results, in a new method Filter::getFilteredData()

  • Improved the Cards Search Tool Filter.js script to ignore punctuation when sorting the filtered ResultSet. With this, Cards like Raznus, Ambassador to the Danians will stay among the other Cards starting with R instead of going to the end of the list

  • Modified all Data Parsers of Cards Search Tool so they can prepare the HTML element where the parsed data would go before actually parsing them. This improved the overall performance by removing a condition previously executed within a loop <_<

  • Updated Changelog Page

Fixed

Structure-related changes
  • Fixed an URL duplication issue that was likely the cause of Hugo considering Episodes’ Listing Pages as Single Pages “randomly”, always failing in a specific condition
Hugo Templates-related changes
  • Fixed an almost unnoticeable an extra space in Site Title
Public Content-related changes

  • Fixed a typo in a keyword about Labyrinth of Linsic in Episode #20, Season #01 (English only)

  • Fixed the Brazilian Voice Actor of Regginald Terrington (ReggieOne) previously defined as Fábio Lucindo while the correct would be Vagner Fagundes, Voice Actor of, among many other roles, Shikamaru Nara in Naruto, Son Gohan in Dragon Ball Kai and Monkey D. Luffy in One Piece

  • Fixed rarity of Skeletal Springs, wrongly defined as Common

  • Fixed rarity of Ribbian, wrongly defined as Rare

  • Fixed rarity of Narfall, wrongly defined as Common (Brazilian Portuguese only)

  • Fixed rarity of Phelphor, wrongly defined as Rare

  • Fixed the i18n entries about Ørth, the Empowerer and his Content Files to fully describe his name instead of just Ørth

  • Fixed an issue with Cards Search Tool that was causing the search results when querying All Cards immediately after searching by a specific Card Type. In this scenario, the first results (by Type) were kept in the dynamically created table and all other results appended to them, messing with the sorting. Even though this is now fixed, in some locales this may give the impression of not fixed. For example: Most of the times, Aa’une is the first Card to be listed because of the two consecutive A in his name, but in Brazilian Portuguese for example, because the article the translates into a, the space after it in te translated version of Cards like The Passage (both versions) make these two Cards to appear before Aa’une instead

  • Fixed the i18n entry about Mount Pillar URL (Brazilian Portuguese only)

  • Fixed an issue with Cards Search Tool that was causing occasional duplication of Aa’une, the Oligarch -OR- one of the Attack Cards, randomly o.O

Image-related changes

  • Added a Big Picture for Takinom, the Shadowknight as part of the Content File Componentization implemented in the main Article about Takinom as a BETA feature

  • Added the following Big Pictures for the new Content Files added as references for the Article about Bodal published in this revision: Lucky Shot and Viperlash (Attacks), Mount Pillar Reservoir and Gloomuck Swamp (Locations) and Orchis Undis (Battlegear)

Miscellaneous changes

  • Updated Sitemap files (except Videos and Mugics)

  • Minified Card Search Tool scripts
