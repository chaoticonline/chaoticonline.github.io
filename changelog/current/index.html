Added

Structure-related changes

  • Added Content Files, Data Files and i18n entries for: Lake Blakeer; The Hive Gallery; The Infectorium; Lore’s Chamber of Recall; Oipont’s Lookout; OverWorld Embassy at Mount Pillar; Queen’s Gate; Mount Pillar Reservoir, Neth’uar’s Foothold; and Mount Pillar Reservoir, Liberated (Locations); Iparu and Kinnianne (Creatures, OverWorld); Ajara and Kolmo (Creatures, Mipedians); Ghundac; Katharaz; Klencka; Klencka, Avenger; Makromil, Nakal; Odu-Bathax, Reservoir Reclaimer; Valanii Levaan; and Yondaf (Creatures, Danians) as counterparts of the references made in the Article about the Mount Pillar published in this revision.

  • Added A Flux too Far as one of the Episodes in which the Forest of Life appeared in the Animation (Brazilian Portuguese only)

Hugo Templates-related changes

  • Added a new possibility for the FrontMatter Parameter shareable named top, allowing Pages to restrict the new Sharing Menu to only appear on the top of the Page, thus preventing it to getting cloned on the bottom. This prevents the User viewing an Article with little content to see the Sharing menu leaving the top area and appearing down below (or vice-versa)

  • Added a <div> wrapping the Like/Dislike icons to improve the stronger grey border styles (see below)

  • Added a condition to use different Site IDs for the Like/Dislike feature if running the site under Hugo Server or Live

Public Content-related changes

  • Added The Hive Gallery as a keyword for Episode #07, Season #01

  • Added The Infectorium as a keyword for Episode #17, Season #01

  • Added Queen’s Gate as a keyword for Episode #37, Season #01

  • Added Iparu as a keyword for Episode #26, Season #01 — shame on you >:(

  • Added Forest of Life as a keyword for Episode #28, Season #01

  • Published Article about Mount Pillar

CSS-related changes

  • Added some margins to the <h4> elements used on Listing Pages to get it a little more centred than Flexbox allowed us

  • From the Article about *Mount Pillar published in this revision and onwards, we’ll add links to the Componentizable Articles “plugged” onto the main version. But because of the links colourization accordingly to the Card Type, a new rule had to be added to keep these texts greyish-black

  • Added some margins to <sup> elements (overall) and a new rule for <sup> elements within paragraphs

  • Added a very small top-margin to the advertisement block of Listing Pages

  • With the new top-margin added to the <sup> elements, the rule positioning these elements on Videos Pages received a negating rule to not misaligned them in there

Removed

Structure-related changes

  • With the revision in all of Google AdSense’s advertisement code-blocks, we ended up reverting their JS deferring, thus removing the necessary routines from our compiled JS code

  • For the Article about Mount Pillar published in this revision, we changed the file structure of Mount Pillar Reservoir, moving it to a subdirectory /mount-pillar and, therefore, the old file reserved for it, as well as its corresponding Data File, have been removed

CSS-related changes

  • Removed the left/right margins of advertisements blocks, allowing them to be horizontally centred

  • Removed some now deprecated rules

Changed

Structure-related changes
  • Adjusted the keyword of Queen’s Gate on Episode #03, Season #02 and Episode #10, Season #03 (Brazilian Portuguese only)
Hugo Templates-related changes

  • Restricted the rendering of the new Sharing Menu’s “clone” to main Articles, this way there’s no need of workaround for the Componentizable Articles, like through FrontMatter Parameters

  • The revision made on Google AdSense’s advertisement code-blocks required the HTML markup in all Pages to be reviewed too

  • Changed the main Template File for Articles to add the HTML attribute class only if there’s an entry tribe in the Data File being processed

  • Applied the same conditioning used for the META Tags title, description and headline for their OpenGraph counterparts

Public Content-related changes

  • Revised all manually added (shortcode) Google AdSense’s advertisement code-blocks in all Content Files they were already added

  • Changed the Data File of Headmaster Ankhyja, complementing the name of the author of its Picture

  • Revised the Brazilian Portuguese translation of Raznus, Ambassador to the Danians

  • With the revision made above, its URI changed and all occurrences have been updated throughout the site

  • Changed the URI pointing to the unpublished Articles about Ikkatosh, Guardian of Nature (Creature) and Song of Stasis (Mugic) in all Content Files they were used so they point directly to URI defined for them, instead of pointing to one of their aliases, strengthening the relation as no Redirection would be needed

  • Improved the writing in the published Article about the Iron Pillar, regarding UnderWorld’s crust

  • Changed the uncertainty writing regarding the Jagged Pillar and the Steam Pillar in the published Article about the Iron Pillar to match with the Article about the Mount Pillar published in this revision

  • Small grammar correction in the published Componentizable Article about Riverlands, Erak’tabb’s Foothold (English only)

  • Updated the Content Files of all published Articles about Location changing the Section Title Game to a more generic Perim, allowing, in a future update, their content to be more related to Locations themselves rather the Card in the Game

  • Updated Changelog Page

CSS-related changes

  • Adjusted the positioning of the new Sharing Menu on Firefox and for mobile devices

  • Moved the stronger grey border to the first child of the Sharing Menu element, this way the border gets slightly shorter without “invading” the padding between the entries of the menu

  • Adjusted the top-margin of the horizontal line of the Sharing Menu’s “clone” at the bottom of the Pages

  • Adjusted the margins of Flavour Text elements (a.k.a Quote) for Locations Pages

  • With the revision in all Google AdSense’s advertisement code-blocks made in this revision (again), we’re experimenting a full width for the wrapper element of its <ins> tags, instead of the previously defined minimum of 250px

  • Adjusted the top-margin of advertisement blocks within the published Articles

  • Increased the thumbnails dimensions on Listing Pages as they were before the previous revision

  • Increased the dimensions of the Locations Card Images

  • With the increase above, the Image Credits positioning had to be adjusted

  • Changed the cursor when hovering the Card Image on the results of the Cards’ Search Tool

  • Because there’s no horizontal border below the entries of the new Sharing Menu on the mobile version, the stronger grey border under the Like/Dislike icons is also removed. But since now there’s another wrapping <div>, the rule affecting this behaviour had to be adjusted

  • Revised the whole style of the Sharing Menu’s “clone” on the mobile version because it was severely misaligned

  • Increased the overall font-size on the mobile version

  • Increased the font-size of <h4> of Listing Pages on the mobile version

  • Increased the bottom margin of Flavour Texts elements (a.k.a. Quote) on the mobile version for when they’re next to a Card Information Box

  • Increased how much space the extended definition of <abbr> elements can occupy on the mobile version

Fixed

Structure-related changes

  • Fixed the URI of Changelog Revision b9c13d9 wrongly pointing to the revision cc4406c (Brazilian Portuguese only)

  • Fixed the URI of Changelog Revisions bb9e032 and fee56ab wrongly pointing to its Brazilian Portuguese translation (English only)

Hugo Templates-related changes

  • Fixed an issue with the Template “Shortcode-Partial” Files for Creatures and Locations in which even if a Data File provided specific information about Image Credits, the default ones would be used instead

  • Fixed an issue introduced in the previous revision in which Locations without a Custom Card Image would display the Portrait fallback instead of the Landscape version

Public Content-related changes

  • Fixed an issue with the Mirror Selection on Episodes Pages because the initial implementation was raising console errors on Articles Pages because, obviously, the needed markup is not present on them

  • Fixed the Brazilian Portuguese translation of the Changelog Revision b9c13d9 that was completely wrong o.O

Image-related changes

  • All Locations Cards existing so far had been updated to reflect the many changes in the template used to create them

  • Added the following Locations Card Images for the Content Files added as references for the Article about Mount Pillar published in this revision: Lore’s Chamber of Recall; Mount Pillar; Mount Pillar Reservoir; Mount Pillar Reservoir, Neth’uar’s Foothold; Mount Pillar Reservoir, Liberated; Oipont’s Lookout; OverWorld Embassy at Mount Pillar; Queen’s Gate; The Hive Galley; and The Infectorium

  • Added Big Pictures for the following Articles created as references for the Article about Mount Pillar published in this revision: Oipont’s Lookout and Queen’s Gate (Locations); Ghundac; Katharaz; Klencka, Avenger; Valanii Levaan; and Yondaf (Creatures)

  • Updated the Big Pictures of Mount Pillar and Mount Pillar Reservoir with versions of better quality

  • Removed a duplicated Card Image of Lake Ken-I-Po, Ritual Site

Miscellaneous changes

  • Added Hugo’s directory /resources to the .gitignore file because its contents are dynamically generated on each update

  • Updated Sitemap Files for Locations, Creatures, Videos and Index

  • Updated minified CSS and JS
