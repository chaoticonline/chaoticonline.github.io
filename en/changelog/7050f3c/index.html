Added

Structure-related changes

  • With the internal structure change (see below), in order to make the Perim Category to work a Page with its own Custom URL had to be added listing all Card Types on it

  • And with the change above, an additional Breadcrumb Trail has been added to all Pages under the Perim Category pointing to the new Page created

  • Added i18n (internationalization) entries for the newly added Perim Listing Page and for the “Share on Facebook/Twitter” hover title change (see below)

  • Added a Hugo’s Compiler setting to prevent automatic pluralization of Section Titles, preventing, for example “Perims”, even though the corresponding i18n entry isn’t pluralized

Hugo Templates-related changes

  • The necessary additions to the Video Template to display the Episode Cast has been created, however, since this still somewhat in BETA stage, it was only implemented in the Episode #01, Season #01

  • Added a more friendly hover text of the Social Icons on the left using the recently added i18n entry “Share on Facebook/Twitter”

Public Content-related changes

  • Added the missing Publishing Date for the following Articles: Tartarek, Ulmar and Riverlands

  • Added the recently identified Voice Actors: Intress (Rachael Lillis), Mezzmarr, Tangath Toborn (Dan Green), Khybon (Kevin Kolack), Lord Van Bloot (David Zen Mansley), Magmon (Eric Stuart), Ulmar (Sean Schemmel) and Wamma (Darren Dunstan)

  • Added a new i18n file with all URLs of all identified Voice Actors’ Profiles. With this, not only the other entry link doesn’t need to be manually filled anymore (see more below) but also it’s not needed to duplicate information in cases in which a Voice Actor portrays multiple characters

  • Following the addition above, all Voice Actors’ Profile URL has been changed in favour of MyAnimeList.com. Although restrict to Animes and regardless the fact “Chaotic” doesn’t appear on the website — since it’s not an Anime per-se — it’s more appropriate for the context (cartoon) -AND- it offers more complete and organized information about the Characters, the Voice Actors they were portrayed by and different productions the Actor participated in, which is not always possible from IMDB or Wikipedia for example. However, since not all Voice Actors work with Animes, IMDB and Wikipedia remains as a fallback

  • Added missing Advertisement Block in the published Article about Tartarek (English only)

  • Added an additional paragraph under the Animation Heading Title in the published Article of about Takinom so the same paragraph on Nauthilax’s Article become more precise

  • Published the Episode #16, Season #01 with Brazilian Portuguese audio

CSS-related changes

  • Added the Tribal Colours for Danians (brown) and Tribeless (light grey) so the Voice Actor box and the URLs of the episodes in which the Creature appear don’t get the default colours and/or without background

  • Added a subtle grey line between commits in the Changelog Page allowing some sort of distinguishing between them

Removed

Structure-related changes
  • Since the i18n entry referring to Episodes as a Tilte isn’t used anymore (see below) it was removed
Hugo Templates-related changes

  • With the internal structure change (see below) there’s no need of a Template File just for the Trivias as this subcategory now shares the same used for the regular Episodes and, therefore, it was removed

  • Removed the possibility of having a Creature/Location with “mixed” rarity in favour of using the Card Shortcode attribute rarity

  • Removed Facebook Analytics. Not only because resources dependable of Facebook slows down the page load but also because the information provided isn’t useful, at least not for now

Changed

Structure-related changes

  • A heavy change in the internal structure has been made. Now instead of having all Perim’s categories of Content Files loose in Hugo’s /content root directory, they’re now grouped under /perim as it always should be. Not only this is more organized but it also allows Section Pages (a.k.a Listing Pages) to have a Custom URL of their own instead of just an alias

  • With the change above, almost all occurrences of Hugo’s variable .Section have been replaced by .CurrentSection, which moves the “pointer” from the beginning of the path to its end

  • The FrontMatter Parameters weight and datePublished have been moved closer to the top where they should be as they represent basic information about the Articles/Pages

  • Updated the FrontMatter Parameters’ comments in the English version of Takinom’s Article with the changes made in this revision

  • With the addition of an i18n file with the Profile URLs of the identified Voice Actors’, the way links are associated to the persons are now automatic. As long as the Voice Actor name is correctly spelt, regardless word casing or accented characters (as they’re treated automatically) -AND- an i18n entry with this cleansed version of the name can be found, the text will be automatically linked

  • Since both Episodes and Trivias are part of the same group now, the i18n entry with the link pointing to the section /episodes has been changed to /videos

Public Content-related changes

  • Grammar correction in the published Article about Tartarek

  • Improved Lord Van Bloot’s “Flavour Text” (Brazilian Portuguese only)

  • In order to display the recently identified Voice Actors (see above), regardless the publishing state of the Articles, since both Episodes and Voice Actor(s) share the same HTML Template, the list of episodes in which those Creatures appear has been filled

  • Improved the paragraph under the Appearances Heading Title, under Animation, not only to simplify the writing but also not to imply that the Creature/Location described in the Article necessarily appear “in person”, after all, they can simply be visually mentioned as a picture, for example, in the Battleboard without effectively being used in a Chaotic battle

  • The video file of Episode #15, Season #01 with Brazilian Portuguese audio has been redone in some parts in order to improve transitions between the remastered, with the dubbed audio and the original audio reused (scenes with only background sounds or just effects, for example) and, therefore, its Page has been updated with the new, updated link

  • Replaced all Breadcrumb Trails and their links from Episodes with Videos in all Content Files, since the former served solely for when Episodes and Trivias couldn’t be of the same group

  • Updated Changelog Page

Fixed

Structure-related changes
  • Fixed an issue with malformed FrontMatter Parameters that should yield runtime errors but prior to the restructuring made in this revision used to be silently ignored >:(
Public Content-related changes

  • Fixed Takinom’s rarity (Brazilian Portuguese only)

  • Fixed a wrong keyword about slurhk’s name in Episode #08, Season #02

  • Fixed the Article about Akkrean which didn’t have the Taxonomy Term rarity (English only)

  • Fixed the video length of Episode #15, Season #01 for the version with Brazilian Portuguese audio which is slightly different of the English one

  • Fixed the details of Episode #15, Season #01 for the version with Brazilian Portuguese audio because this version is not in Englis (obviously) but also doesn’t have any subtitle as previously described

Miscellaneous changes

  • Updated minified CSS
