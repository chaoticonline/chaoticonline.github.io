Added

Structure-related changes

  • Added an i18n for Sílvio Giraldi as the second Brazilian Portuguese Voice Actor of Xaerv

  • Added Faduli Costha as the second Brazilian Portuguese Voice Actor of Siado

  • Added Sidney Lilla as the third Brazilian Portuguese Voice Actor of Khybon

  • Added Luiz Antônio Lobue as the Brazilian Portuguese Voice Actor of Valanii Levaan, although, at least for now, only for Episode #28, Season #01

  • Added Mauro Castro as Voice Actor of Prince Mudeenu in Episode #18, Season #01 (Brazilian Portuguese only)

  • With the recent identification of the second Brazilian Portuguese Voice Actor of Siado, the entry appearances in his Data File has been added

  • Added i18n for the texts Copy and Copied as well for the “title” in the Sharing Menu

Hugo Templates-related changes
  • Added an additional conditional to the Partial Template of Google Analytics so the reports don’t take into account local accesses during development
Public Content-related changes

  • Added Cast for Episode #28, Season #01

  • With the recent identification of the Brazilian Portuguese Voice Actors of Siado and Valanii Levaan, the Shortcode markup needed to display the information have been added to their Content Files

  • Published Episode #28, Season #01 with Brazilian Portuguese audio

CSS-related changes
  • Added a negative top-margin to the Creatures’ Voice Actors’ Box so it becomes aligned to the standardized participation phrase as well as it gets further away from Facebook Comments Plugin, improving the visibility in Pages in which Creature didn’t appear in many Episodes (i.e. Valanii Levaan)

Removed

Public Content-related changes

  • Removed Faduli Costha as the Brazilian Portuguese Voice Actor of Vidav in Episode #27, Season #01, as the studio responsible, disrespecting the Voice Actor, didn’t dub the character’s scream/grunt

  • Removed Carlos Silveira as the Brazilian Portuguese Voice Actor of Xaerv in Episode #18, Season #01 as the studio responsible, disrespecting the Voice Actor, didn’t dub the character’s scream/grunt

  • Removed a duplicated keyword for Siado in Episode #22, Season #01 Content File

Changed

Structure-related changes
  • Because, apparently, Khybon doesn’t have one single “official” Brazilian Portuguese Voice Actor, all note entries in his Data File have been changed specifying the Episode in which that Actor voiced him
Hugo Templates-related changes
  • Improved the Video Player implementation on Episodes’ Pages
Public Content-related changes

  • Improved the writing in the published Article about the Iron Pillar, removing a pluralization after the expandable definition (<abbr>) about peroxide bonds (Brazilian Portuguese only)

  • Improved the writing in the published Article about the Iron Pillar, replacing the word peroxy with peroxide in the expandable definition (<abbr>) (English only)

  • Expanded the Section Title heading the Componentizable published Article about Mount Pillar Reservoir, Liberated in order to indicate that the expression Liberated refers to the Reservoir instead of some other random sub-location

  • Confirmed Carlos Silveira as the first Brazilian Portuguese Voice Actor of Xaerv

  • Moved the Google AdSense’s Advertisement Block to below the results of Cards’ Search Tool

  • Moved the “funny text” in Cards’ Search Tool’s Page below the repositioned Google AdSense’s Advertisement Block

  • Updated Changelog Page

CSS-related changes

  • Changed the way possible notes regarding a Voice Actor appear within the Box to the right of a Creature’s Appearances list in the Animation. Now they’re considered a block element, breaking to a new line thus providing more space for the Voice Actors’ Name

  • Improved the styles of Episodes’ Servers Selection, mirrors icons (see below) and Video Player on Episodes’ Pages, without the need of palliative solutions between Google Chrome and Mozilla Firefox

  • Improved the display appearance of an URL in the text field to be copied in the Sharing Menu by adding the “dots”

Fixed

Structure-related changes
  • Fixed Title and URI in the published Componentizable Article about Nauthilax, Tainted (Brazilian Portuguese only)
Hugo Templates-related changes
  • Fixed a semantic issue on Episodes’ Pages in which multiple occurrences of the Loading Icon existed by changing its identification through an ID with a class
Public Content-related changes

  • Fixed Flavour Text (a.k.a. “Quote”) in the unpublished Article about Iflar, Crown Prince (Brazilian Portuguese only)

  • Fixed the FrontMatter Parameter data in the Componentizable published Article about Glacier Plains, M’arrillian Melting Camp which was pointing to the Data File designated to M’arrillian Heat Cannon (Brazilian Portuguese only)

  • Fixed the link to The Hive Gallery which was wrongly pointing to The Infectorium in the published Article about Mount Pillar (Brazilian Portuguese only)

  • Fixed an issue with Tune of Xerium Content File which was preventing the Mugic Tribe to be properly translated

  • Fixed The Training Grounds Region and Initiative

  • Fixed a templating issue with Changelog Page (English only)

CSS-related changes

  • Fixed the rule of possible notes regarding a Voice Actor

  • Fixed an issue with the alignment icon of Vimeo in comparison to the Video Player introduced in the previous revision

Image-related changes
  • Replaced the Episodes’ mirrors icons images with SVG definitions, providing unlimited resizing capabilities, improving performance and allowing a fine vertical alignment of Vimeo icon, which is taller than the YouTube icon
Miscellaneous changes
  • Updated minified CSS and JS files
