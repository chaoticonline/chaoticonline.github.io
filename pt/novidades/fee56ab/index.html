Adições

Estruturais

  • Adicionado um ícone SVG de ampulheta para representar a Aliança Temporária de algumas Criaturas com os M’arrillians

  • Adicionado o Parâmetro de FrontMatter isGroup aos arquivos das Páginas de Listagem /perim/creatures/_index.<LANG>.md, afinal eles também são uma “categoria de categorias”

Relacionados ao Hugo

  • Adicionada margem superior ao Bloco de Anúncios na Página da Ferramenta de Busca de Cartas

  • Implementado o conceito do Hugo de Template-Base permitindo organizar melhor todos os Arquivos de Template por fracionar grandes conteúdos em blocos menores por arquivo. Com essa abordagem foram definidos seis blocos:

    • category** para definir o atributo HTML id usado na tag <section> principal
    • subcategory para classes CSS adicionais a serem usadas na tag <section> principal
    • section para definir o valor da tag <h2>
    • title para definir o valor da tag <h3>
    • content para o conteúdo principal (doh)
    • pagination usado especificamente pelos Templates das Páginas de Listagem para construir os Links de Paginação

  • Implementado os blocos necessários parte do Template-Base do Hugo nos Arquivos de Template das Páginas de Listagem e das Páginas Individuais, tanto os Padrões quanto para Vídeos, refatorando e re-indentando sempre que possível — ex: evitando múltiplas declarações else para construir uma estrutura por criar previamente a dita estrutura com seus valores-padrão

  • Implementada uma forma de exibir os Blocos de Anúncio em todas as Páginas de Listagem exceto às de Listagem de Tribos independente do viewport reduzindo o número de resultados a serem exibidos em 1 (um) e removendo a classe de controle que o ocultava. Porque estamos carregando um resultado a menos, Páginas as quais todas as entradas possuem uma Imagem Grande — preenchendo assim todas as colunas disponíveis — carregarão de forma ligeiramente mais rápida, pois haverá uma imagem a menos para ser feito o download

Relacionados ao Conteúdo Publicado

  • Adicionado um segundo Bloco de Anúncios do tipo “in-ads” no Artigo publicado sobre o Tangath Toborn já que ele é bem longo e comporta sem forçar demais

  • Publicado Episódio #25, Temporada #01 com áudio em Português Brasileiro

Relacionados aos Estilos do Site

  • Adicionadas todas as regras CSS necessárias como contraparte do ícone SVG de ampulheta adicionado

  • Adicionadas algumas regras adicionais aos Blocos de Anúncios como contraparte da implementação melhorada para tê-los em meios aos resultados de Página de Listagem (ver acima)

Relacionados aos Estilos do Site

Alterações

Estruturais
  • Já que os próprios Desenvolvedores do Hugo reconhecem que a forma como eles manipulam arquivos JSON não é a mais adequada, consumindo bastante memória, decidimos integrar a lista de Episódios nos quais as Criaturas/Locais (e, no futuro, Equipamentos de Batalha e Mugics também) aparecem na Animação aos Data Files com a maioria de suas informações (nome, raridade…). Mesmo que isso signifique duplicar a mesma informação por todos os arquivos localizados (ex: um para cada idioma), pelo menos podemos tentar reduzir o uso de memória, principalmente durante o desenvolvimento
Relacionados ao Hugo

  • Melhorada a implementação do Template Parcial para o Google AdSense adicionando bordas para quando o site esteja rodando através do Servidor do Hugo ou se nosso Parâmetro de Desenvolvimento esteja setado como true no arquivo config.yaml -e- para adicionar automaticamente o ID de Cliente vindo do recentemente adicionado opção de configuração quando o devido Atributo/Parâmetro não tiver sido setado na chamada do Shortcode

  • Enquanto implementando o conceito de Template-Base do Hugo, nós separamos as Páginas de Listagem de Cartas da de Listagem de Vídeos, reduzindo a lógica necessária para cada Template, mesmo que a maior parte do código entre eles seja idêntica

  • Ajustada a cor principal adicionada diretamente no Template Parcial de Cabeçalho como parte da metodologia Critical Path para combinar com a cor da Otimização de Contraste de Cores feita há algumas revisões atrás

  • Refatorado o Template Parcial Perim responsável por construir a Página Individual dos Artigos sobre as Cartas removendo o Scratch do Hugo além de melhorar o código em geral onde possível

  • Refatorados os Templates Parciais dos Shortcodes de Criaturas/Locais responsáveis por construir as Caixas de Informações das Cartas por aplicar parte do conceito de Template-Base do Hugo e, em alguns casos, reusando lógicas relativamente longas múltiplas vezes

  • Refatorados os Templates Parciais dos Shortcodes de Criaturas/Locais (TV) responsáveis por construir a lista de Episódios nos quais eles aparecem e, no caso das Criaturas, a caixa com seus Dubladores

Relacionados ao Conteúdo Publicado

  • Com a melhora no Template Parcial do Google AdSense *TODOS* os Artigos publicados não-componentizáveis foram atualizados para simplificar suas chamadas ao Shortcode

  • Atualizada Página de Novidades

Relacionados aos Estilos do Site

  • Com a melhora na implementação das Páginas de Listagem (ver acima) todas as classes CSS nomeadas .listing foram convertidas para um ID como em #listing

  • Com a melhora na implementação das Páginas de Listagem (ver acima) todas as classes CSS nomeadas .page.videos foram convertidas para um ID como em #videos
